# 角色过往经历与情绪触发系统

> 为20位主角补充过往经历（故事开始前发生的事）和情绪触发系统（什么会触发什么情绪）

---

## 一、过往经历与行为的关系

```
过往经历 → 形成信念/创伤 → 触发点 → 影响当前行为

童年经历 → 核心信念（"世界是怎样的"）
关键创伤 → 行为烙印（"在类似情境下会如何反应"）
形成信念 → 价值判断（"什么是对的/重要的"）
```

---

## 二、情绪触发系统

**核心思路**：情绪不是独立字段，而是场景的修饰因子

```
行为预判 = 基础设定 × 情绪状态 × 场景类型
```

**情绪类型**：
- **平静**：默认状态，按常规行为模式
- **愤怒**：窄化注意力，冲动攻击，突破常规
- **悲伤**：降低动力，被动退缩，行动力下降
- **焦虑**：过度思考，犹豫拖延，留下破绽
- **恐惧**：放大风险，保守防御，逃避
- **兴奋**：高估收益，冒险冲动，过度自信

## 四、使用方法

### 场景创作时

1. **读取基础设定**：哲学流派、心理特征、核心矛盾
2. **读取过往经历**：理解行为根源
3. **确定成长阶段**：当前处于成长弧光的哪个位置
4. **确定情绪状态**：是否有触发点被激活
5. **输出行为预判**：第一反应 / 后续行动 / 内心独白
