# 同步哲学设定和社会结构

## 四、关键设定速查

### 血牙血脉

- **血脉**：熊血脉（守护之力）★
- **来源**：第一章正设定稿确认
- **能力**：强大体魄、血脉守护、防御能力

### AI入侵本质

- **本质**：无形无相、悄无声息、随机选取
- **真相**：血脉异常体的真正原因（角色不知，中期揭示）

### 世界观核心原则

- **核心原则**：无正邪，只有立场
- **核心主题**：「我是谁」身份认同贯穿始终
- **感情线原则**：跨势力/跨种族，悲剧为主，少数救赎

| 蛰伏时代 | 隐忍、积累、希望、暗流 | 灰蓝、暗绿 | 雨、雾、山 |
