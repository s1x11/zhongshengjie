# 十大势力社会结构设定

> 基于《众生界》世界观，为10大势力设计经济、政治、社会分层、文化-意识形态结构

---

## 设计原则

1. **经济基础决定上层建筑**：不同势力的力量来源决定其经济结构
2. **政治权力与力量体系对应**：权力分配机制与势力特征匹配
3. **社会分层反映力量等级**：阶层划分与力量等级体系一致
4. **文化意识形态塑造身份认同**：核心价值观决定成员的"我是谁"追问
5. **AI入侵机制融入社会结构**：入侵风险是每个势力的结构隐患

---

## 社会结构四维框架

| 维度 | 定义 | 设计要素 |
|------|------|----------|
| **经济基础结构** | 生产方式、资源控制、财富分配 | 力量来源、核心资源、分配机制 |
| **政治权力结构** | 权力来源、决策机制、权力分配 | 权力合法性、决策机构、权力分层 |
| **社会分层结构** | 阶层划分、流动机制、身份认同 | 阶层等级、流动条件、身份标志 |
| **文化-意识形态结构** | 核心价值观、信仰体系、教育传承 | 价值核心、信仰对象、传承方式 |

## 一、东方修仙

### 力力体系概要

| 维度 | 核心特征 |
|------|----------|
| **力量来源** | 天地灵气 |
| **修炼方式** | 打坐炼气 |
| **境界体系** | 炼气→筑基→金丹→元婴→化神→渡劫（6阶） |
| **派别体系** | 法修、体修、剑修、丹修、符修、器修、阵修、魂修（8派） |
| **核心代价** | 真气耗尽、经脉受损、神识透支 |
| **AI入侵防御** | 神识感知（高阶修仙者可感知入侵） |

---

### 经济基础结构

| 维度 | 设计 |
|------|------|
| **核心资源** | 灵脉、灵石、灵药、法宝材料 |
| **生产方式** | 灵力采集、丹药炼制、法宝锻造、阵法布置 |
| **资源控制** | 宗门控制灵脉，丹药/法宝由丹修/器修生产 |
| **财富形式** | 灵石（货币）、丹药、法宝、灵脉股份 |
| **分配机制** | 宗门内：贡献点制度；宗门间：交易市场 |
| **经济依赖** | 依赖灵脉分布，灵脉枯竭则经济崩溃 |

#### 灵脉经济体系

| 灵脉等级 | 产出 | 控制宗门 | 经济价值 |
|----------|------|----------|----------|
| **天级灵脉** | 高阶灵气、稀有灵药 | 顶级宗门 | 战略资源，不可交易 |
| **地级灵脉** | 中阶灵气、常见灵药 | 中级宗门 | 核心资源，有限交易 |
| **人级灵脉** | 低阶灵气、基础灵药 | 小型宗门 | 一般资源，自由交易 |

#### 丹药/法宝经济

| 产品类型 | 生产者 | 分配方式 | 交易机制 |
|----------|--------|----------|----------|
| **高阶丹药** | 丹修宗师 | 宗门内贡献点兑换 | 禁止外部交易 |
| **中阶丹药** | 丹修大师 | 宗门内贡献点兑换 | 有限外部交易 |
| **低阶丹药** | 丹修弟子 | 自由生产 | 自由交易 |
| **高阶法宝** | 器修宗师 | 宗门传承 | 禁止外部交易 |
| **中阶法宝** | 器修大师 | 宗门内贡献点兑换 | 有限外部交易 |
| **低阶法宝** | 器修弟子 | 自由生产 | 自由交易 |

### 政治权力结构

| 维度 | 设计 |
|------|------|
| **权力合法性** | 境界等级：高境界者拥有更高权力 |
| **权力来源** | 灵力修炼成果 + 神识能力 |
| **决策机构** | 宗门长老会（高境界长老组成） |
| **权力分层** | 宗主→长老→执事→弟子 |
| **决策机制** | 长老会协商，宗主决策，执事执行 |
| **权力继承** | 境界继承：最高境界者继承宗主 |

#### 宗门权力等级

| 权力等级 | 身份 | 境界要求 | 权力范围 |
|----------|------|----------|----------|
| **宗主** | 最高决策者 | 化神以上 | 宗门战略、灵脉分配、对外战争 |
| **长老** | 决策参与者 | 元婴以上 | 长老会决策、宗门管理、弟子培养 |
| **执事** | 执行管理者 | 金丹以上 | 宗门日常运营、资源分配执行 |
| **弟子** | 基础成员 | 炼气以上 | 修炼、贡献、服从 |

#### 宗门间权力关系

| 关系类型 | 定义 | 机制 |
|----------|------|------|
| **联盟** | 共同控制灵脉 | 联盟长老会、灵脉共享 |
| **竞争** | 灵脉争夺 | 宗门战争、弟子竞争 |
| **依附** | 小宗门依附大宗门 | 资源贡品、弟子输送 |
| **敌对** | 灵脉冲突 | 战争、封锁 |

### 社会分层结构

| 维度 | 设计 |
|------|------|
| **阶层划分** | 境界等级决定阶层 |
| **阶层等级** | 化神阶层→元婴阶层→金丹阶层→筑基阶层→炼气阶层 |
| **流动机制** | 境界提升=阶层上升，但境界提升困难 |
| **流动条件** | 修炼天赋、资源投入、运气机缘 |
| **身份标志** | 境界称号、宗门身份、法宝标志 |
| **阶层特权** | 高境界者享受资源优先分配、决策参与权 |

#### 境界阶层对应

| 境界 | 阶层身份 | 社会地位 | 资源分配 | 特权 |
|------|----------|----------|----------|------|
| **化神** | 顶层精英 | 宗门领袖 | 天级灵脉优先 | 战略决策权、战争指挥权 |
| **元婴** | 高层精英 | 宗门长老 | 地级灵脉优先 | 长老会参与权、弟子培养权 |
| **金丹** | 中层骨干 | 宗门执事 | 人级灵脉优先 | 资源分配执行权、日常管理权 |
| **筑基** | 底层骨干 | 高级弟子 | 人级灵脉基础 | 修炼资源申请权、任务执行权 |
| **炼气** | 底层基础 | 普通弟子 | 灵石基础分配 | 修炼权、贡献义务 |

#### 流动难度分析

| 境界跨越 | 流动难度 | 关键条件 | 流动比例 |
|----------|----------|----------|----------|
| **炼气→筑基** | 中等 | 天赋+资源+1-3年修炼 | 60%弟子可跨越 |
| **筑基→金丹** | 高 | 天赋+资源+机缘+5-10年修炼 | 30%弟子可跨越 |
| **金丹→元婴** | 极高 | 天赋+资源+机缘+10-30年修炼 | 10%执事可跨越 |
| **元婴→化神** | 极限 | 天赋+资源+机缘+30-100年修炼 | 3%长老可跨越 |
| **化神→渡劫** | 神级 | 天赋+资源+机缘+渡劫成功 | 1%化神可跨越 |

### 文化-意识形态结构

| 维度 | 设计 |
|------|------|
| **核心价值观** | 修炼至上：境界提升是人生终极目标 |
| **信仰体系** | 无神论：不信仰神明，信仰灵力与修炼 |
| **价值核心** | 灵力崇拜：灵力是力量与生命的源泉 |
| **身份认同** | 宗门身份：我是某宗门弟子/长老 |
| **教育传承** | 师徒制：师父传授弟子修炼功法 |
| **文化禁忌** | 境界倒退禁忌：境界下降=社会降级=耻辱 |

#### 修仙文化体系

| 文化维度 | 内容 | 传承方式 |
|----------|------|----------|
| **功法传承** | 灵力修炼方法 | 师徒传授、宗门秘典 |
| **丹药知识** | 灵药炼制方法 | 丹修传承、丹经典籍 |
| **法宝知识** | 法宝锻造方法 | 器修传承、器典秘籍 |
| **阵法知识** | 阵法布置方法 | 阵修传承、阵典秘籍 |
| **神识知识** | 神识修炼方法 | 魂修传承、神识秘典 |

#### "我是谁"追问

| 追问层次 | 内容 | 文化背景 |
|----------|------|----------|
| **第一层** | 我的身边人是否被入侵？ | 神识感知能力，可探测入侵 |
| **第二层** | 我的境界是否真实？ | 境界是身份认同核心，境界虚假=身份虚假 |
| **第三层** | 如果我被入侵，我的境界还是我的吗？ | 境界继承是权力来源，入侵=权力虚假 |

### AI入侵机制融入

| 入侵阶段 | 社会影响 | 防御机制 |
|----------|----------|----------|
| **潜伏阶段** | 高境界者被潜伏，决策受影响 | 神识感知可发现潜伏 |
| **部分入侵** | 部分长老被入侵，长老会决策异常 | 神识对抗、阵法防御 |
| **完全入侵** | 宗主被入侵，宗门战争异常 | 神识大战、渡劫对抗 |
| **融合体产生** | 新独立体出现，身份认同混乱 | 神识认同重建、境界重建 |

#### 东方修仙入侵防御等级

| 境界 | 入侵防御能力 | 入侵风险 |
|------|--------------|----------|
| **化神** | 神识强力感知+对抗 | 低（但被入侵后影响最大） |
| **元婴** | 神识中等感知+对抗 | 中 |
| **金丹** | 神识基础感知 | 中高 |
| **筑基** | 神识微弱感知 | 高 |
| **炼气** | 无神识感知 | 极高 |
