# 时间线追踪（Timeline Tracking）

> 记录《众生界》故事时间流逝、事件顺序、角色年龄变化，确保时间线一致性。

---

## 使用说明

### 时间线记录原则

| 原则 | 说明 |
|------|------|
| **绝对时间** | 以"第X天"形式记录，从故事开始为第1天 |
| **相对时间** | 记录距上一事件的天数 |
| **季节状态** | 标注当前季节，便于环境描写一致性 |
| **角色年龄** | 重要角色的年龄/阶段变化 |

### 时间单位换算

| 单位 | 说明 |
|------|------|
| **天** | 最小单位，记录日常事件 |
| **旬** | 10天，适合中期跨度 |
| **月** | 30天，适合阶段总结 |
| **年** | 365天，适合长跨度事件 |

## 时间线检查清单

每章创作后检查：

- [x] 时间流逝是否合理（距上一章天数） → 第1章有十年跨度
- [x] 事件顺序是否正确（因果关系） → 血脉暴走→追踪→灭族，因果正确
- [x] 季节描写是否一致 → 深秋雨夜、晨曦朝阳，一致
- [x] 角色年龄是否同步增长 → 血牙10-13岁→23岁，十年增长
- [x] 白天/黑夜时间是否合理 → 黎明→雨夜→朝阳→十年后，合理

## 常见时间线问题

| 问题类型 | 示例 | 修正方法 |
|---------|------|---------|
| **时间倒流** | 第10章说"三天后"，第11章却回到"当天" | 统一时间线，明确标注 |
| **季节跳跃** | 第5章秋天，第6章突然春天 | 保持季节连续性 |
| **年龄不增长** | 故事跨度1年，角色年龄不变 | 按时间比例增长 |
| **昼夜混乱** | 同一场景内出现白天→黑夜→白天 | 检查场景时间线 |
