# 资源账本（Resource Ledger）

> 记录《众生界》重要资源的流向，防止资源凭空出现/消失，确保连续性。

---

## 使用说明

### 资源类型

| 类型 | 说明 | 追踪粒度 |
|------|------|---------|
| **灵石** | 修炼资源、货币 | 数量、来源、消耗 |
| **法器** | 武器、防具、辅助法器 | 名称、品质、状态 |
| **丹药** | 治疗、突破、辅助 | 名称、数量、有效期 |
| **功法** | 修炼功法、战技 | 名称、品级、残篇/完整 |
| **伤势** | 角色当前伤势 | 部位、程度、恢复进度 |
| **人脉** | 重要关系、承诺 | 对象、性质、是否兑现 |

### 更新原则

- **新增**：必须注明来源（获得方式、章节）
- **消耗**：必须注明去向（使用方式、章节）
- **转折**：必须注明原因（损毁、丢失、赠送）
