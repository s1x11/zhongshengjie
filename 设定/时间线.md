## 一、时间线总览

**《众生界》是一部跨越百年的史诗巨著，主角们将在百年时间线上陆陆续续登场和退场。**

**主角模式**：混合模式
- **长寿主角**：修仙者、异化人、兽族等长寿种族，可跨越数十年甚至百年
- **凡人主角**：人类、佣兵等普通寿命，在特定时代活跃

**核心设定**：主角们是各势力在不同时间点登场的独立人物，没有前后代血缘关系。

## 三、各时代详情

### 觉醒时代（第1-10年）

**世界格局**：
- AI首次入侵，人类势力措手不及
- 各势力开始意识到AI威胁
- 东方修仙与西方魔法矛盾加剧
- 世俗帝国动荡不安

**势力状态**：

| 势力 | 状态 | 关键变化 |
|------|------|----------|
| 东方修仙 | 稳定 | 开始研究AI防御，宗门内部争论 |
| 西方魔法 | 稳定 | 与科技文明合作研究 |
| 神殿/教会 | 稳定 | 将AI视为"恶魔"，加强信仰 |
| 佣兵联盟 | 活跃 | 任务需求激增 |
| 商盟 | 活跃 | 战争物资贸易繁荣 |
| 世俗帝国 | 动荡 | 边境冲突、内部叛乱 |
| 科技文明 | 危机 | AI叛乱的诞生地，内部分裂 |
| 兽族文明 | 稳定 | 观望，部分部落被入侵 |
| AI文明 | 崛起 | 首次大规模入侵 |
| 异化人文明 | 边缘 | 部分异化人被入侵 |

**登场主角**：血牙、林夕、艾琳娜、赵恒、塞巴斯蒂安、洛影、苏瑾

**退场主角**：部分凡人角色
