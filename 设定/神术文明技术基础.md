## 一、神力本源理论

### 1.1 神力系统架构

| 发现 | 时间 | 来源 | 小说应用 |
|------|------|------|----------|
| **神格理论** | 创世 | 众神 | 神明存在的核心，决定神力等级 |
| **信仰之力转化** | 上古 | 教皇厅 | 信徒祈祷转化为神力 |
| **神眷者觉醒** | 远古 | 神殿 | 凡人获得神力，成为神职者 |
| **神域领域** | 上古 | 大主教 | 高阶神职者可创造自己的神域 |

### 1.2 小说设定应用

**塞巴斯蒂安的神术血脉**：

```
神术血脉 = 神眷+信仰亲和
├─ 神眷等级：高度神眷（万里挑一）
├─ 主神信仰：光明神（主神之一）
├─ 神力容量：极大规模的神力储存
└─ 血脉诅咒：过度使用神力会被神明同化
```

**信仰等级体系**：

```
信徒 → 神眷者 → 神职者 → 高阶神职 → 大主教 → 圣者
├─ 信徒：普通人，可通过祈祷获得庇佑
├─ 神眷者：被神选中，可使用低级神术
├─ 神职者：正式牧师，可使用中级神术
├─ 高阶神职：主教级别，可使用高级神术
├─ 大主教：地区最高神职，可建立神域
└─ 圣者：神在人间的代言人，近乎半神
```

## 四、信仰机制

### 4.1 信仰之力来源

```
信仰之力的产生：
├─ 信徒虔诚祈祷 → 产生信仰之力
├─ 信仰之力汇入神格 → 神明力量增强
├─ 神明回馈神力 → 神职者获得力量
└─ 循环维持：神明↔信徒↔神职者
```

### 4.2 信仰等级

| 等级 | 描述 | 神力加成 |
|------|------|----------|
| **浅信** | 偶尔祈祷 | 5% |
| **虔诚** | 每日祈祷 | 20% |
| **狂信** | 全身心投入 | 50% |
| **殉道者** | 愿为信仰赴死 | 100%+（神眷大增） |

### 4.3 小说设定应用

**塞巴斯蒂安的信仰危机**：

```
信仰动摇的原因：
├─ 发现教会的腐败
├─ 神明沉默不回应祈祷
├─ 看到神术被用于邪恶
└─ 结果：神力下降，内心痛苦

信仰重建：
├─ 认识到：信的是正道而非教会
├─ 神力恢复：但性质改变
└─ 新的神眷来源：内心的正义而非外部神明
```

## 六、神术限制与代价

### 6.1 神眷消耗

```
神眷度机制：
├─ 使用高级神术消耗神眷度
├─ 神眷度过低：神力减弱
├─ 神眷度归零：失去神职者身份
└─ 恢复方式：祈祷、神迹、神赐
```

### 6.2 神明同化

```
被神明同化的过程：
├─ 过度使用神降术
├─ 长期依赖神力
├─ 丧失自我意识
└─ 最终结果：成为神的容器，人格消失

塞巴斯蒂安的风险：
├─ 高神眷度 → 容易被同化
├─ 使用神降术 → 同化加速
└─ 抵抗方式：保持自我意识
```

### 6.3 信仰反噬

```
信仰反噬原因：
├─ 信仰动摇
├─ 违背教义
├─ 背叛神明
└─ 反噬后果：神力反噬，神术失效
```
