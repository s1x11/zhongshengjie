# 主角哲学/心理基调设定

> 基于20个哲学流派，为众生界20位主角设计哲学关切、命运观、世界观、心理特征

---

## 设计原则

1. **势力归属决定哲学底色**：不同势力的世界观塑造主角的哲学起点
2. **入侵状态决定核心矛盾**：AI入侵是"我是谁"最极端的实践场
3. **命运彼岸决定哲学终点**：彼岸不是力量提升，而是精神归宿
4. **哲学流派与人物成长弧光对应**：从哲学起点到哲学终点的蜕变

## 20位主角哲学/心理基调映射

### 第一组：传统势力（5人）

#### 1. 林夕（东方修仙）

| 维度 | 设计 |
|------|------|
| **哲学流派** | 东方哲学（道家） |
| **核心关切** | "我是无我/大我" |
| **哲学起点** | 小我：执着于"证明自己"，背负宗门期望 |
| **哲学终点** | 大我：不争之争，真正的强者不需要证明 |
| **命运观** | 建构论：命运是修炼的选择，但最终领悟"无为而无不为" |
| **世界观** | 一元论：天地灵气与自我合一 |
| **心理特征** | 踏实但有野心，内敛但有爆发力 |
| **核心矛盾** | 野心 vs 无我，证明自己 vs 不争之争 |
| **行为模式** | 默默修炼，关键时刻爆发；从"我要证明"到"我不需要证明" |
| **"我是谁"追问** | 第一层：身边人是否被入侵？（修仙者神识可感知） |
| **成长弧光** | 小我 → 修炼执念 → 顿悟 → 大我 → 不争之争 |
