# 行为预判模板

> 基于设定推导角色在特定场景下的行为

---

## 模板使用方法

### 快速推导

```
步骤1：确定角色和场景
步骤2：查阅该角色的过往经历和情绪触发
步骤3：确定当前成长阶段
步骤4：确定当前情绪状态
步骤5：输出行为预判
```

### 推导公式

```
行为预判 = 哲学底色 × 心理特征 × 核心矛盾 × 情绪状态 × 场景类型
```

---

## 行为预判模板

```markdown
## [角色名] · [场景类型]

**当前阶段**：[成长弧光的位置]
**当前情绪**：[平静/愤怒/悲伤/焦虑/恐惧/兴奋]
**触发点激活**：[是/否，什么触发点]

### 行为预判

| 维度 | 内容 | 推导依据 |
|------|------|----------|
| **第一反应** | [表情、动作、第一句话] | [心理特征/过往经历] |
| **后续行动** | [分步骤列出] | [行为模式/核心矛盾] |
| **内心独白** | [内心在想什么] | [哲学关切/核心矛盾] |

### 特殊情况

| 情绪变化 | 触发条件 | 行为变化 |
|----------|----------|----------|
| [情绪A] | [触发条件] | [行为变化] |
| [情绪B] | [触发条件] | [行为变化] |
```

## 核心场景类型（16种）

| 场景 | 核心要素 | 常见行为关注点 |
|------|----------|----------------|
| **战斗** | 生死、代价、保护 | 战斗风格、不畏死程度、保护谁 |
| **情感** | 表达、羁绊、牺牲 | 是否主动、是否脆弱、付出程度 |
| **悬念** | 真相、隐瞒、揭示 | 是否追求真相、如何处理信息 |
| **社交** | 关系、利益、立场 | 是否主动社交、利益优先还是情感优先 |
| **冲突** | 立场、代价、选择 | 如何站队、是否妥协 |
| **危机** | 时间、压力、抉择 | 压力下的表现、是否冷静 |
| **背叛** | 信任、愤怒、代价 | 反应激烈程度、是否原谅 |
| **牺牲** | 代价、选择、价值 | 为谁牺牲、牺牲的意义 |
| **失去** | 悲伤、承受、重生 | 如何面对失去 |
| **获得** | 喜悦、警惕、代价 | 如何面对获得 |
| **诱惑** | 欲望、克制、代价 | 是否能抵抗 |
| **抉择** | 两难、代价、信念 | 如何做选择 |
| **重逢** | 情感、变化、陌生 | 如何面对故人 |
| **对峙** | 压力、底牌、破绽 | 如何应对压力 |
| **谈判** | 利益、底线、让步 | 谈判风格 |
| **独处** | 真实、脆弱、反思 | 独处时的状态 |
```

## 情绪状态对照表

| 情绪 | 认知影响 | 行为倾向 | 典型表现 |
|------|----------|----------|----------|
| **平静** | 理性思考 | 按常规行为模式 | 按设定行为 |
| **愤怒** | 窄化注意力 | 冲动、攻击、突破常规 | 战意外放、不惜代价 |
| **悲伤** | 降低动力 | 被动、退缩 | 沉默更深、独自一人 |
| **焦虑** | 过度思考 | 犹豫、拖延、留下破绽 | 冒险行动、错误决策 |
| **恐惧** | 放大风险 | 保守、防御、逃避 | 退缩、逃避 |
| **兴奋** | 高估收益 | 冒险、冲动、过度自信 | 轻敌、过度行动 |

## 示例：血牙在战斗场景

### 血牙 · 战斗场景

**当前阶段**：十年复仇期
**当前情绪**：平静（默认状态）
**触发点激活**：否

### 行为预判

| 维度 | 内容 | 推导依据 |
|------|------|----------|
| **第一反应** | 冷静评估敌人实力，血脉气息压制 | 血脉能力：危险时自动压制气息 |
| **后续行动** | 不主动攻击，观察破绽，关键时刻一击必杀 | 行为模式：沉默深沉，关键时刻爆发 |
| **内心独白** | "记住每一张脸，记住每一个破绽" | 复仇执念 |

### 特殊情况

| 情绪变化 | 触发条件 | 行为变化 |
|----------|----------|----------|
| **愤怒** | 提及佣兵联盟、看见剥皮行为 | 战意爆发，不畏死，一换一也要杀 |
| **悲伤** | 想起母亲/父亲 | 沉默更深，但战意不减 |
| **焦虑** | 身边重要之人（花姬）有危险 | 冒险行动，留下破绽 |

## 示例：林夕在情感场景

### 林夕 · 情感场景

**当前阶段**：小我期（尚未顿悟）
**当前情绪**：平静
**触发点激活**：否

### 行为预判

| 维度 | 内容 | 推导依据 |
|------|------|----------|
| **第一反应** | 克制表达，不主动示弱 | 心理特征：内敛但有爆发力 |
| **后续行动** | 默默关注，用行动而非语言表达 | 行为模式：默默修炼，关键时刻爆发 |
| **内心独白** | "情感是羁绊，但也是人的证明" | 核心矛盾：证明自己 vs 不争之争 |

### 特殊情况

| 情绪变化 | 触发条件 | 行为变化 |
|----------|----------|----------|
| **愤怒** | 宗门被侮辱、重要之人被威胁 | 不再克制，战意外放 |
| **悲伤** | 失去重要之人 | 沉默更深，独自修炼 |

## 批量生成命令

如果需要为所有角色生成行为预判，可以使用以下模板批量生成：

```
角色：[角色名]
场景：[场景类型]
阶段：[成长阶段]
情绪：[情绪状态]

输出格式：
- 第一反应：[内容]
- 后续行动：[内容]
- 内心独白：[内容]
- 依据：[哲学底色/心理特征/核心矛盾/过往经历]
```
