# 众生界创作支持系统使用指南 v5.0

---

## 一、系统概述

本系统为《众生界》创作提供**三大数据库**支持：

| 数据库 | 存储位置 | 内容 | 用途 |
|--------|----------|------|------|
| 小说设定库 | Qdrant | **143条**（势力、角色、力量体系、时代） | 世界观查询 |
| 创作技法库 | Qdrant | **904条**（11维度技法） | 创作指导 |
| 案例库 | Qdrant | **256,083条**（跨题材标杆案例） | 参考借鉴 |

**核心文件**：
- `workflow.py` — 统一工作流入口 ★
- `graph_visualizer.py` — 知识图谱可视化生成
- `technique_graph_visualizer.py` — 技法图谱可视化生成

**可视化**：
- `knowledge_graph.html` — 知识图谱交互式可视化
- `technique_graph.html` — 技法关系图可视化

## 三、统一工作流 (workflow.py)

### 命令行使用

```bash
cd D:\动画\众生界\.vectorstore

# 查看统计
python workflow.py --stats

# 检索小说设定
python workflow.py --search-novel "熊血脉 守护之力"
python workflow.py --search-novel "东方修仙" --entity-type "势力"

# 检索创作技法
python workflow.py --search-technique "战斗代价" --dimension "战斗冲突维度"

# 检索案例
python workflow.py --search-case "部落战斗 血脉燃烧" --scene "战斗场景"

# 列出场景/作家
python workflow.py --list-scenes
python workflow.py --list-writers
```

### Python API

```python
from workflow import NovelWorkflow

workflow = NovelWorkflow()

# 检索小说设定
results = workflow.search_novel("熊血脉", entity_type="力量派别")

# 检索创作技法
results = workflow.search_techniques("战斗代价", dimension="战斗冲突维度")

# 检索案例
results = workflow.search_cases("部落战斗", scene_type="战斗场景")

# 获取统计
stats = workflow.get_stats()
```

## 九、常见问题

### Q: 检索结果不相关？

A: 尝试调整查询词或使用更具体的维度过滤。

### Q: Docker Qdrant连接失败？

A: 确保Docker容器运行中：
```bash
docker ps | grep qdrant
docker start qdrant  # 如果未运行
```

### Q: 如何重新同步数据？

A: 按顺序执行：
```bash
# 1. 从设定文件同步到知识图谱
python merge_settings_to_graph.py
python merge_worldview_to_graph.py

# 2. 清理重复
python cleanup_knowledge_graph.py

# 3. 同步到向量库
python rebuild_qdrant_docker.py --all

# 4. 重新生成可视化
python graph_visualizer.py
python technique_graph_visualizer.py
```

### Q: 血牙的血脉是什么？

A: **熊血脉（守护之力）**。已在所有数据源同步修正。
