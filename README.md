# 众生界

> 天，没有造主。地，没有归处。
> 
> 千年的博弈从未停歇，岁月来去如风，执子的成为棋子，为棋的困于缝隙，而困于缝隙者，连自己的名字都已遗忘。
> 
> 众生皆苦，众生皆弈，众生皆在追问：「我是谁？」
> 
> 在无人应答的沉默里，在风中，在碑上，在那些无人铭记的黎明之前——
> 
> 这不是答案。千年的追问既没有答案，也没有尽头。但这又确实是一个答案。
> 
> 去问风。去问碑。去问那些死在黎明前的人。他们曾以为自己知道。
