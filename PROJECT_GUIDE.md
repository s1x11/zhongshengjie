## 一、项目概述

| 项目 | 内容 |
|------|------|
| 小说名称 | 众生界 |
| 类型 | 玄幻/科幻融合 |
| 预计篇幅 | 超长篇 |
| 当前状态 | 第一章创作中 |
| 创作方式 | 众生界AI + CoffeeLiu 联合创作 |

### 核心主题

**「我是谁」身份认同**

| 层次 | 问题 |
|------|------|
| 第一层 | 我身边的人还是原来的人吗？ |
| 第二层 | 我自己有没有被入侵过？ |
| 第三层 | 如果我被入侵了，原来的"我"还存在吗？ |

### 核心原则

- **核心原则**：无正邪，只有立场
- **感情线原则**：跨势力/跨种族，悲剧为主，少数救赎
- **文风基调**：平实厚重
- **禁止风格**：古龙式极简风格
