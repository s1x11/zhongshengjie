# 众生界知识图谱标准化方案

> 本文档定义知识图谱的数据结构、关系类型和更新流程

---

## 一、系统架构

本系统采用**双库分离架构**：

| 系统 | 集合名称 | 内容 | 用途 |
|------|----------|------|------|
| 小说设定库 | `novel_settings` | 势力、角色、力量体系、时代、事件 | 世界观关系网络 |
| 创作技法库 | `writing_techniques` | 10维度技法 | 创作检索指导 |

**核心说明**：知识图谱仅包含小说设定，不含创作技法。

---

## 二、实体类型

| 类型 | 说明 | 示例 |
|------|------|------|
| **角色** | 小说人物 | 林夕、血牙、零 |
| **势力** | 组织/文明 | 东方修仙、AI文明 |
| **派系** | 势力内分支 | 紫云峰、白月阁 |
| **事件** | 发生的事情 | 觉醒之夜、镜的复仇 |
| **时代** | 时间段 | 觉醒时代、风暴时代 |
| **力量体系** | 修炼体系 | 修仙、魔法、兽力 |
| **力量派别** | 力量体系分支 | 剑修、丹修、法修 |

## 四、数据更新流程

### 4.1 标准更新流程

```bash
# 1. 编辑源文件（设定/*.md、创作技法/*.md）

# 2. 同步到向量数据库（双库同步）
cd D:\动画\众生界\.vectorstore
python sync_to_vectorstore_v3.py

# 3. 重建知识图谱（仅小说设定）
python rebuild_knowledge_graph_v2.py

# 4. 生成可视化
python graph_visualizer.py          # 知识图谱
python technique_graph_visualizer.py # 技法关系图
```

### 4.2 使用统一工作流

```python
from workflow import NovelWorkflow

workflow = NovelWorkflow()

# 查看统计
stats = workflow.get_stats()

# 检索设定
char = workflow.get_character("林夕")
techs = workflow.search_techniques("战斗代价")

# 获取图谱
graph = workflow.get_knowledge_graph()
```
