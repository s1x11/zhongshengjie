# 小说结构十法

十种经典小说结构技巧，涵盖悬念设置、情节安排、线索设计、收尾技法等核心创作方法。

---

## 一、横切悬念，倒叙事件法

### 技法说明

> **在小说首段设置提挈全篇、笼罩全文的悬念，故意给读者造成疑团，以激起读者兴趣读下去。**

| 要素 | 说明 |
|------|------|
| **核心** | 切入悬念 → 倒叙事件 |
| **目的** | 避免平铺直叙，激起读者兴趣 |
| **效果** | 读者非读下去不可 |

### 操作步骤

```
1. 首段设置悬念（切入悬念）
   ↓
2. 给读者造成疑团
   ↓
3. 激起读者兴趣
   ↓
4. 倒叙事件（解释悬念）
   ↓
5. 读者持续阅读
```

### 经典案例：《一双明亮的眼睛》

**悬念设置**：
- 夜，墨黑，伸手不见五指
- "我"第一次去生产大队，路生，没月亮，没星星，没手电
- 碰到一个社员，把"我"引去了
- 沿途他准确指点："前头有条沟！""左边是口塘！""里面住着咱们队长"

**倒叙揭示**：
- 第二天清早，"我"看到一个强壮的中年人挑着桶向堰塘走来
- 一瞄，眼怎么瞎了？
- 他蛮不在乎地下塘挑水
- "我"大吃一惊喊是塘，他回头眨眼说："你不就是我昨夜给你引路的同志吗？"

**读者疑问**（悬念效果）：
- 他是个瞎子吗？
- 为什么能那么利索地引人走夜晚？
- 为什么他比有眼睛的人的"眼睛"还明亮？

### 使用要点

| 要点 | 说明 |
|------|------|
| **悬念要足够吸引** | 能笼罩全文，提挈全篇 |
| **疑团要合理** | 不能太荒诞，要能解释 |
| **倒叙要清晰** | 解释悬念时逻辑要清楚 |
| **节奏要控制** | 悬念后不要拖太久才解释 |

## 二、意料之外，情理之中法

### 技法说明

> **创作结构要巧，首先要"出其不意"，但更重要的是所叙述的情节必须在情理之中。**

| 要素 | 说明 |
|------|------|
| **核心** | 出其不意 + 情理之中 |
| **第一步** | 创造意想不到的东西 |
| **更重要** | 与人物性格发展合拍，合乎客观规律，合乎生活逻辑 |

### 卓别林名言

> "我总是力图以新的方法来创造意想不到的东西。假如我相信观众预料我会在街上走，那我便跳上一辆马车去。"

### 经典案例：《麦琪的礼物》（欧·亨利）

**故事结构**：

```
圣诞节 → 恩爱夫妇准备互赠礼物
    ↓
妻子：看到丈夫有祖传金表，但没有表链
    → 剪掉自己最珍爱的金色长发，卖了买表链
    ↓
丈夫：看到妻子有美丽金发，但缺少名贵梳子
    → 卖掉祖传的金表，买了梳子
    ↓
两人碰面：
- 丈夫拿着新表链，表没有了
- 妻子拿着新梳子，长发没有了
    ↓
夫妻俩只好凄然相对而笑
```

**出其不意**：
- 两人都卖掉了自己最珍爱的东西
- 买的礼物对方都用不了了

**情理之中**：
- 夫妻恩爱，超过了对"金发"、"表链"的感情
- "金钱第一"的资本主义世界，对下层小人物只能是辛辣结局

### 使用要点

| 要点 | 说明 |
|------|------|
| **意外性** | 要超出读者预料 |
| **合理性** | 必须符合人物性格发展 |
| **逻辑性** | 合乎客观规律和生活逻辑 |
| **不是荒诞** | 不是臆造的，要曲折离奇又理所当然 |

## 三、淡化情节，形散神聚法

### 技法说明

> **表面看没有出其不意的情节，没有激烈的矛盾冲突，平铺直叙，一直是淡淡的气氛。但在这平铺直叙中，带有涌袭心灵的感情；在这淡淡的凄凉中，带有一种说不出的人生韵味。**

| 要素 | 说明 |
|------|------|
| **核心** | 形散神聚，无结构的结构 |
| **表面** | 平铺直叙，淡淡气氛 |
| **内在** | 涌袭心灵的感情，人生韵味 |
| **特点** | "无情"更有情，无声胜有声 |

### 经典案例：《拾麦穗》（张洁）

**故事情节**：
- 农村小姑娘，家里很穷
- 每年夏天割麦时，她挽个篮子到打过麦的麦地里拾麦穗
- 卖麦芽糖的老汉来了
- 别的孩子用拾的麦穗与老汉换糖吃，小女孩舍不得
- 老汉便常常免费敲糖给她吃
- 别人笑她，说她嫁给这个老汉算了
- 她对此不怎么懂，老汉也觉得没什么
- 大家开开玩笑，以后也没再提起

**高潮**（淡淡的）：
- 老汉没再来卖糖时
- 她在村头等着，等着……
- 她在等什么呢？

**内在韵味**：
- 不仅仅是为了吃老汉的糖
- 有说不清的味
- 有一种淡淡哀恋
- 一种人与人之间能相互沟通的情绪

### 使用要点

| 要点 | 说明 |
|------|------|
| **不要激烈冲突** | 保持淡淡的气氛 |
| **情感要真挚** | 平铺直叙中带有涌袭心灵的感情 |
| **内在精神编织** | 用一种内在精神编织"情结"文体 |
| **韵味要足** | 带有人说不出的人生韵味 |

## 四、一箭双雕，一点两面法

### 技法说明

> **利用一个"道具"，不只让一个角色，而使许多角色与这个道具有关系；不只让一方，而是让矛盾的双方都与这个道具打交道。从这个道具身上挖掘人物心灵世界，揭露生活本质。**

| 要素 | 说明 |
|------|------|
| **核心** | 一个道具，多方关系 |
| **目的** | 挖掘人物心灵，揭露生活本质 |
| **效果** | 一箭双雕，深刻揭示 |

### 经典案例：《二十先令的银币》（斯丹·巴斯托）

**道具**：20 先令的银币

**人物关系**：

```
马斯顿太太（有钱）
    ↓
故意把银币放进一套衣服里
    ↓
让仆人弗斯戴克太太送衣服到洗衣店
    ↓
交代：送洗之前，得把衣服口袋掏一掏
    ↓
弗斯戴克太太（穷人，丈夫瘫痪，急需钱）
    ↓
发现银币后的思想斗争：
- 可以给丈夫买水果、买烟、加一瓶酒
- 可以买几件必须添置的衣服
    ↓
经过艰苦斗争后，没用这个钱
    ↓
颤悠悠地交还了
```

**一箭双雕**：
1. **展现了世界穷人物质的贫穷**
2. **暴露了富人精神上的无耻**

**深刻揭示**：
- 这个世界人与人之间的关系是多么残酷
- 马斯顿太太的目的：希望弗斯戴克瞒下银币并花掉，然后叫她交钱，证明穷人穷得卑劣下作，她自己才是高尚的

### 使用要点

| 要点 | 说明 |
|------|------|
| **道具选择** | 要有象征意义，能承载主题 |
| **多方关系** | 让矛盾双方都与道具打交道 |
| **挖掘心灵** | 从道具身上挖掘人物心灵世界 |
| **揭露本质** | 通过道具揭露生活本质 |

## 五、偶然中必然，必然中偶然法

### 技法说明

> **学会在生活中发现偶然中隐藏着的必然性，学会在写小说时运用这种偶然中的必然性。它能引发读者寻根盘底地、津津有味地追读下去，而且能揭示生活中不易发现的本质意义。**

| 要素 | 说明 |
|------|------|
| **核心** | 偶然中的必然，必然中的偶然 |
| **效果** | 引发读者追读，揭示本质意义 |
| **要求** | 节节都隐蔽着偶然与必然的关系 |

### 经典案例：《项链》（莫泊桑）

**情节链条**：

```
借项链
  ↓ （偶然：为了参加上流社会晚会）
  ↓ （必然：小职员妻子想得人欢心，被人艳羡）
失项链
  ↓ （偶然：归途上丢失）
  ↓ （必然：乐极生悲，虚荣的代价）
赔项链
  ↓ （必然：不得不承受经济压力）
  ↓ （偶然：偏偏是她借的项链）
还项链债务
  ↓ （必然：十年艰苦努力）
  ↓ （偶然：刚还完债才发现）
发现项链是假玩意
  ↓ （必然：一夜风头得到十年辛酸）
  ↓ （必然：片刻虚荣换来半生痛苦）
```

**偶然中的必然**：
- 借项链是偶然，但想攀附上流社会是必然
- 失项链是偶然，但虚荣心导致疏忽是必然
- 发现项链是假的是偶然，但资本主义社会虚伪是必然

### 使用要点

| 要点 | 说明 |
|------|------|
| **发现偶然** | 在生活中发现偶然中隐藏的必然 |
| **运用偶然** | 在写作时运用偶然中的必然 |
| **节节隐蔽** | 每个情节都隐蔽着偶然与必然 |
| **揭示本质** | 揭示生活中不易发现的本质意义 |

## 六、银丝串珠，数点一线法

### 技法说明

> **用荒诞的、超现实的、纯粹主观想象的"抽象"物编织描绘。从表面看是分离的、意识流的、不易理解的。但内部互为联系，相辅相成，彼此烘托，合成一体。**

| 要素 | 说明 |
|------|------|
| **核心** | 笔断意不断，藕断丝连 |
| **表面** | 分离的、意识流的、不易理解的 |
| **内在** | 互为联系，相辅相成，彼此烘托 |
| **适用** | 现代派小说，反映精神危机 |

### 经典案例：《……以后》（亨利·斯莱萨）

**主题**：核战争恐怖

**四段结构**：

| 段落 | 人物 | 情节 | 反映 |
|------|------|------|------|
| **博士** | 记忆学教授 | 核战后失业，改教"如何忘记" | 核战惨状令人不敢回忆 |
| **律师** | 杀人犯律师 | 核战后人口减少 90%，男女比例 800:1，杀人犯极刑是与 18 个女人结婚 | 人口结构剧变 |
| **商人** | 服饰商 | 核战后变种生育，65% 孩子有两个头，帽子服饰畅销 | 变种生育 |
| **酋长** | 土著酋长 | 白人躲到孤岛，仪器检测辐射毒，酋长吃了白人肉染上辐射毒 | 辐射毒无处幸免 |

**内在联系**：
- 四段无连贯情节
- 但以四个不同的荒诞侧面，反映了世界一些阶层在核战争后的心态
- 用当代与未来采用他们的人生观穿起来

### 使用要点

| 要点 | 说明 |
|------|------|
| **表面分离** | 可以是分离的、意识流的 |
| **内在联系** | 内部要互为联系，相辅相成 |
| **彼此烘托** | 各部分要彼此烘托，合成一体 |
| **人生观串联** | 采用作者的人生观把当代与未来穿起来 |

## 七、明线暗线——双环连套法

### 技法说明

> **运用文中两个一明一暗的线索，平行交叉，双环连套，从一个人物引出一个人物，从一个故事引出一个故事，不仅使两个人物、两个故事发生密切的关系，而且不断丰富人物性格，推动主题思想深化。**

| 要素 | 说明 |
|------|------|
| **核心** | 一明一暗，平行交叉，双环连套 |
| **效果** | 丰富人物性格，深化主题思想 |
| **结构** | 从一个人物引出一个人物，从一个故事引出一个故事 |

### 经典案例：《药》（鲁迅）

**明线**：
```
华老栓的儿子小栓得了痨病
  ↓
人们说吃了血馒头就好了
  ↓
老栓等到清早衙门杀人，拿着洋钱去买血馒头
  ↓
匆匆拿回家给小栓吃
  ↓
小栓最终被痨病夺去生命
```

**暗线**：
```
老栓去取血馒头那天清早
  ↓
被杀的囚犯是革命者夏瑜
  ↓
小栓就是吃的他的血馒头
  ↓
但这剂"药"并没把小栓的病治好
```

**双环连套**（结局坟场）：
```
小栓的墓和夏瑜的墓错落地挨在一起
  ↓
小栓的母亲、夏瑜的母亲都在给儿子上坟、烧纸
  ↓
两条线在坟场上交汇
```

**主题思想**：
- 愚昧的群众享用革命者的鲜血，不是医治病苦的良药
- 资产阶级脱离群众的革命，不是疗救中国社会的良药

### 使用要点

| 要点 | 说明 |
|------|------|
| **明线清晰** | 明线要让读者清楚看到 |
| **暗线隐蔽** | 暗线要逐步揭示 |
| **最终交汇** | 两条线要在某个点交汇 |
| **主题深化** | 通过双环连套深化主题 |

## 八、欲扬先抑和欲抑先扬法

### 技法说明

> **把自己准备着力表现的人物，不妨先压一压，就如伸出去打人的拳头，先缩一缩。这样，击出去，更有力；而准备贬低的人物，则不妨先让他"威风威风"，然后，让他从"威风岭"上掉下来，便"摔"得更惨。**

| 技法 | 说明 | 比喻 |
|------|------|------|
| **欲扬先抑** | 先压一压，再崛起 | 拳头先缩一缩，击出去更有力 |
| **欲抑先扬** | 先让他威风，再摔下来 | 从"威风岭"上掉下来，摔得更惨 |

### 欲扬先抑案例：《我的第一个上级》（马烽）

**先抑**（表面看）：
- 县农建局的田副局长
- 衣冠不整，披个烂棉袄
- 无精打采，疲疲塌塌，慢慢吞吞
- 简直不象个领导样子

**后扬**（异峰崛起）：
- 在海门决堤抢险过程中
- 熟谙全局，胸有成竹，果敢勇猛
- 不顾 1954 年防汛中患上的、一直没有好的、十分严重的关节炎
- 身先士卒地带头在大风大浪中搏斗
- 一直到战胜险情才收兵

**效果**：塑造了一个活生生的社会主义英雄人物

### 欲抑先扬案例：《水浒》武松醉打蒋门神

**先扬**（蒋门神）：
- 描绘得如何厉害，如何了不得
- 很多武艺高强的人都不是他的对手

**后抑**（武松出场）：
- 几个回合，蒋门神就趴下了

**效果**：
1. 衬托了武松威武
2. 把蒋门神欺软怕硬的虚弱本质揭露无遗

### 使用要点

| 要点 | 说明 |
|------|------|
| **抑要到位** | 压得够低，扬起来才有力 |
| **扬要突然** | 异峰崛起，出人意料 |
| **转折要自然** | 不能太突兀，要有铺垫 |
| **两种可交叉使用** | 欲扬先抑和欲抑先扬可以互辅互补 |

## 九、盆中藏月，以小见大法

### 技法说明

> **用这个技法的小说，一般题材单纯，场景单一，人物较少，情节相对来讲也比较平淡。但如何从单纯的题材和不长的篇幅里塑造丰满的形象和挖掘出较深的主题，就得"盆中藏月，以小见大"了。**

| 要素 | 说明 |
|------|------|
| **核心** | 以小见大，一叶报春 |
| **特点** | 题材单纯，场景单一，人物较少，情节平淡 |
| **目的** | 从单纯题材和短篇幅里塑造丰满形象，挖掘较深主题 |
| **比喻** | 大海能容下皎明的月亮，小小的一盆水也能容下皎明的月亮 |

### 经典案例：《藤椅》（叶文玲）

**故事情节**：
```
中学教师杨健领回学校发的新藤椅
  ↓
全家都感到高兴
  ↓
高兴之后，大家想：新藤椅往哪儿放呢？
  ↓
三代同堂、老幼六口住的 15 平方米房间里
  ↓
现在是连一把椅子也挤不进去了
  ↓
老杨只好婉惜地将藤椅退给学校
```

**作品特点**：
- 一无传奇色彩
- 二无复杂的情节
- 就这么平凡人小事
- 就这么平铺直叙
- 就这么淡淡然，凄凄然

**震撼力**：
- 解放三十年了，一直勤勤恳恳干教育事业的杨老师一家
- 为什么还窝在蜗牛似的小房里？！
- 是"四人帮"十年的"愚民"政策
- 是"白卷大王"散发的精神污染
- 是不是应该清醒清醒，尽快地、实实在在地清除这些"愚民"的余毒和"白卷"的偏见！

### 使用要点

| 要点 | 说明 |
|------|------|
| **题材单纯** | 选择平凡的题材 |
| **挖掘深意** | 从平凡中挖掘深刻主题 |
| **小中见大** | 一叶报春，盆中藏月 |
| **平淡中震撼** | 平铺直叙中震撼读者的心 |

## 十、余音绕梁，三日不绝法

### 技法说明

> **故事的开头好，发展好，但最后一个音符，最后一抹油彩，最后一个标点处理若不当，也会虎头蛇尾，归之失败。因为这不仅是作品的落脚处，还需把作品最精粹的地方展现出来。**

| 要素 | 说明 |
|------|------|
| **核心** | 收尾技巧，余音绕梁 |
| **重要性** | 最后一个音符、油彩、标点处理 |
| **风险** | 处理不当会虎头蛇尾 |
| **目的** | 把作品最精粹的地方展现出来 |

### 收尾要点

| 要点 | 说明 |
|------|------|
| **最后一个音符** | 结尾的基调要与全文一致 |
| **最后一抹油彩** | 结尾的画面要给人留下印象 |
| **最后一个标点** | 结尾的节奏要恰到好处 |
| **作品落脚处** | 结尾是作品的最终归宿 |
| **展现最精粹** | 把作品最好的地方在结尾展现 |

### 收尾方式建议

| 方式 | 说明 | 适用 |
|------|------|------|
| **悬念式** | 留下悬念让读者思考 | 开放式结局 |
| **呼应式** | 与开头呼应，形成闭环 | 结构完整 |
| **升华式** | 主题升华，意境提升 | 有深意的作品 |
| **留白式** | 不说破，让读者体会 | 含蓄作品 |
| **震撼式** | 结尾震撼，印象深刻 | 需要冲击力的作品 |

### 使用要点

| 要点 | 说明 |
|------|------|
| **不要虎头蛇尾** | 结尾要与开头、发展相配 |
| **展现最精粹** | 把作品最好的地方在结尾展现 |
| **余音绕梁** | 让读者读完后还能回味 |
| **处理得当** | 最后一个音符、油彩、标点都要精心处理 |
