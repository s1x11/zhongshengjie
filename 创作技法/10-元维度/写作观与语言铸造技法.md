# 写作观与语言铸造技法

关于写作兴趣、想象力、语言境界、叙事观点等元层面创作技法。

---

## 一、写作兴趣培养

### 1.1 兴趣的本质

> **兴趣是作家写作的要素之一，兴趣是一种狂热**

| 要点 | 说明 |
|------|------|
| **狂热性质** | 不是普通喜欢，而是狂热 |
| **内在驱动** | 来自内心深处，非外在强迫 |
| **持续性** | 能支撑长期写作 |

### 1.2 兴趣培养方法

| 方法 | 说明 |
|------|------|
| **扩大生活面** | 多维度体验生活 |
| **扩大关心面** | 关心更广泛的人和事 |
| **充实知识** | 积累知识和常识 |
| **养成观察习惯** | 观察人与事 |
| **沉思内省** | 深入思考自己 |

## 二、语言铸造三境界

### 2.1 三境界详解

| 境界 | 特点 | 说明 | 案例 |
|------|------|------|------|
| **第三境界** | 日常用语直摘入文 | 意象稀淡疏滥 | "他很难过，哭了" |
| **第二境界** | 过滤日常语言 | 合乎题材角色 | "泪水无声滑落"（符合人物身份） |
| **第一境界** | 塑建作者自己语言 | 含作者独有色彩、韵律、音调、气味、语法、句式、组合、语气 | 属于这位作家的独有语气世界 |

### 2.2 第一境界特征

> **第一境界 = 属于这位作家的独有语气世界的塑建**

| 特征 | 说明 |
|------|------|
| **独特色彩** | 能识别出是某位作家的作品 |
| **独特韵律** | 句子节奏有别于他人 |
| **独特音调** | 文字读起来有独特感觉 |
| **独特气味** | 作品有独特的氛围感 |
| **独特语法** | 有自己的语法习惯 |
| **独特句式** | 偏好某种句式结构 |
| **独特组合** | 词语组合方式独特 |
| **独特语气** | 整体语气风格独特 |

### 2.3 语言进阶路径

```
第三境界（日常用语）
    ↓ （大量阅读 + 大量写作）
第二境界（过滤语言）
    ↓ （塑建风格 + 持续打磨）
第一境界（独有语气世界）
```

### 2.4 语言运用原则

| 原则 | 说明 |
|------|------|
| **自己看得懂** | 基本要求，不能自己都不懂 |
| **不妨晦涩，但要有目的** | 不可为晦涩而晦涩 |
| **可出现无法纳入文法标准的语言** | 如疯子独白、故意混乱 |
| **一般叙述应能被文法剖析** | 常规叙述需符合文法 |

## 三、叙事观点锻炼

### 3.1 两种观点类型

| 观点类型 | 关注点 | 问题 |
|---------|--------|------|
| **技巧观点** | 叙事技术 | 作者采用哪种叙事观点？ |
| **实质观点** | 思想感情 | 作者想表达什么思想感情？ |

### 3.2 技巧观点细分

| 问题 | 说明 |
|------|------|
| **采用哪种叙事观点？** | 第一人称/第三人称/全知视角/限知视角 |
| **对观点的约束力有多少？** | 视角切换是否自由 |
| **作者介入的程度？** | 作者是否直接评论 |

### 3.3 实质观点内涵

| 内涵 | 说明 |
|------|------|
| **人生观点** | 作者自成系统的人生观点 |
| **生命体认** | 对生命的理解和体验 |
| **思想感情** | 想表达于作品的思想、感情 |
| **潜意识运用** | 隐藏在背后的潜意识 |

### 3.4 叙事观点选择建议

| 作品类型 | 推荐视角 | 原因 |
|---------|---------|------|
| **个人成长** | 第一人称 | 真实感强 |
| **宏大叙事** | 第三人称全知 | 视野广阔 |
| **悬疑推理** | 第三人称限知 | 保留悬念 |
| **多线交织** | 多人物视角切换 | 全面展现 |

## 四、文字运用核心技法

### 4.1 文字虚实论

| 类型 | 性质 | 用途 | 写法 |
|------|------|------|------|
| **实文字** | 快速、直接 | 场景跳换、节奏快 | 直白叙述 |
| **虚文字** | 慢速、意象 | 情感抒发、氛围营造 | 用比喻、象征、暗示 |

### 4.2 虚文字写法

**虚的文字要抓起来写**，使用多种手法：

| 手法 | 说明 | 案例 |
|------|------|------|
| **明喻** | 用"像"等直接比喻 | "她的眼睛像星星" |
| **暗喻** | 不用"像"的比喻 | "她是夜空中最亮的星" |
| **主观比喻** | 带主观色彩的比喻 | "那笑容暖得让人想哭" |
| **客观比喻** | 客观描述的比喻 | "那笑容如春日阳光" |
| **象征** | 用具体事物表抽象 | "用秋天象征离别" |
| **暗示** | 不直接说，让读者体会 | "叶子黄了，她也走了" |
| **侧面** | 不从正面写 | 通过他人反应写主角 |

### 4.3 文字节奏控制

| 节奏 | 适用场景 | 技法 |
|------|---------|------|
| **快** | 场景跳换、动作戏 | 短句、实词为主 |
| **慢** | 情感抒发、氛围营造 | 长句、虚词、比喻 |

### 4.4 文字七种写法

| 写法 | 说明 | 适用 |
|------|------|------|
| **深深的写** | 深入刻画 | 重要人物/场景 |
| **浅浅的写** | 轻描淡写 | 次要内容 |
| **浓浓的写** | 情感浓郁 | 高潮部分 |
| **淡淡的写** | 情感克制 | 离别、回忆 |
| **大大的写** | 大笔挥洒 | 大场景 |
| **细细的写** | 细腻描摹 | 细节特写 |
| **灵活的写** | 根据内容调整 | 全文 |

## 五、文字与人物关系

### 5.1 心笔人物关系

> **文字运用虽然是笔随心转，可是心却要随人物而转**

| 关系 | 说明 |
|------|------|
| **笔随心转** | 文字受作者心境影响 |
| **心随人物转** | 作者心境受人物影响 |
| **连带关系** | 三者互相影响 |

### 5.2 常见错误

| 错误 | 表现 | 后果 |
|------|------|------|
| **个性凌驾人物** | 把自己的个性强加在人物头上 | 人物不真实 |
| **语言不随人物** | 所有人物说话一个腔调 | 人物没区分度 |
| **心境不随场景** | 悲伤场景用欢快语言 | 情感不协调 |

### 5.3 正确做法

| 做法 | 说明 |
|------|------|
| **心随人物转** | 根据人物性格调整语言 |
| **语言合乎角色** | 什么身份说什么话 |
| **心境随场景变** | 根据场景调整情感色彩 |

## 六、生活与题材关系

### 6.1 生活决定论

> **小说的语言是完全同生活打成一片的**

| 关系 | 说明 |
|------|------|
| **多广的生活** | 就多广的题材选择度 |
| **多广的题材** | 就需要多广的生活来支持 |

### 6.2 生活积累方法

| 方法 | 说明 |
|------|------|
| **接触各色人等** | 各种年龄、身份、职业的人 |
| **体验多样生活** | 不同场景、不同经历 |
| **储备口语** | 听别人说话并贮藏 |
| **观察细节** | 养成观察习惯 |

### 6.3 生活语言运用

| 要点 | 说明 |
|------|------|
| **生活语言为基础** | 小说文字从生活语言产生 |
| **广博性强** | 生活性非常强 |
| **根据事件选择** | 看不同事件决定用具体或抽象语言 |
| **考虑时空** | 考虑特定时空条件 |

## 七、快速调用指南

### 7.1 语言境界自检

```
语言境界检查：
□ 我的语言是否只是日常用语直摘？（第三境界）
□ 我的语言是否已过滤并合乎角色？（第二境界）
□ 我的语言是否有独特色彩/韵律/气味？（第一境界）
□ 读者能否识别出是我的作品？
□ 我的作品是否有独特语气世界？
```

### 7.2 叙事观点选择流程

```
1. 确定作品类型（个人/宏大/悬疑/多线）
   ↓
2. 选择叙事观点（第一人称/第三人称/多视角）
   ↓
3. 确定约束力（视角切换是否自由）
   ↓
4. 确定作者介入程度（是否直接评论）
   ↓
5. 明确实质观点（想表达什么思想感情）
```

### 7.3 虚实文字运用表

| 场景类型 | 推荐 | 写法 |
|---------|------|------|
| 动作戏 | 实文字 | 短句、快速、直接 |
| 情感戏 | 虚文字 | 比喻、象征、暗示 |
| 场景跳换 | 实文字 | 快速过渡 |
| 氛围营造 | 虚文字 | 慢速、意象丰富 |
| 人物对话 | 自然语言 | 合乎角色身份 |

### 7.4 语言运用原则检验

```
语言运用检查清单：
□ 自己是否看得懂？
□ 晦涩是否有目的？
□ 是否为晦涩而晦涩？（如果是，删掉）
□ 疯子独白等特殊场景可以违反文法
□ 一般叙述是否能被文法剖析？
```

### 7.5 生活积累计划

| 频率 | 任务 | 目标 |
|------|------|------|
| **每天** | 听不同人说话并记录 | 储备对话素材 |
| **每周** | 接触新场景/新人 | 扩大生活面 |
| **每月** | 读不同体裁作品 | 学习不同语言风格 |
| **每年** | 体验新生活方式 | 深化人生体认 |

### 7.6 注意事项

| ❌ 禁止 | ✅ 替代 |
|--------|--------|
| 把个性凌驾人物头上 | 心随人物而转 |
| 为晦涩而晦涩 | 晦涩要有目的 |
| 只写熟悉生活 | 不断扩大生活面 |
| 日常用语直摘 | 过滤并塑建风格 |
| 忽视实质观点 | 技巧与实质并重 |

## 八、作家语言风格养成计划

### 8.1 短期目标（1-3 个月）

| 目标 | 具体做法 |
|------|---------|
| **达第二境界** | 过滤日常语言，合乎题材角色 |
| **熟练虚实运用** | 掌握快慢节奏控制 |
| **培养观察习惯** | 每天记录观察到的细节 |

### 8.2 中期目标（3-12 个月）

| 目标 | 具体做法 |
|------|---------|
| **形成初步风格** | 有自己的语言偏好 |
| **熟练多种写法** | 深/浅/浓/淡灵活运用 |
| **扩大生活面** | 接触更多人和事 |

### 8.3 长期目标（1-3 年）

| 目标 | 具体做法 |
|------|---------|
| **达第一境界** | 塑建独有语气世界 |
| **风格稳定** | 读者能识别作品 |
| **语言成熟** | 虚实得当，节奏自如 |
