# 创作流程技法

> 适用：史诗级小说完整创作流程
> 整合：所有维度技法的流程化应用

---

## 一、创作流程总览

```
史诗级小说创作流程：

第一阶段：构思与世界观（10%时间）
├── 核心公理确立
├── 世界观推导
├── 势力格局设计
└── 主题定位

第二阶段：大纲与结构（15%时间）
├── 多线叙事设计
├── 人物小传创建
├── 冲突链条设计
└── 章节大纲规划

第三阶段：章节创作（60%时间）
├── 场景设计
├── 对话创作
├── 氛围营造
└── 悬念设计

第四阶段：修订与精修（15%时间）
├── 全文检查
├── 技法对照
├── 禁止项清理
└── 节奏调整
```

---

## 二、第一阶段：构思与世界观

### 2.1 核心公理确立

**步骤**：

```
Step 1：确定世界观核心公理（1-3条）
├── 公理必须不可打破
├── 公理是世界底层规则
└── 所有设定从公理推导

Step 2：检验公理
├── 公理是否足够底层？
├── 公理之间是否一致？
├── 公理能否推导完整世界观？
└── 公理是否不可被角色突破？

Step 3：记录公理
├── 写入世界观推导模板
└── 作为所有设定的检验标准
```

**输出物**：
- 世界观推导模板（见99-创作模板/世界观推导模板.md）

### 2.2 世界观推导

**步骤**：

```
Step 1：力量体系推导
├── 力量来源（与公理关系）
├── 力量层级
├── 力量代价
└── 力量限制

Step 2：势力格局推导
├── 势力形成原因
├── 势力发展阶段差异
├── 势力历史恩怨
└── 势力利益冲突

Step 3：地理环境推导
├── 地理与公理关系
├── 地理对生存的影响
├── 地理对文化的影响
└── 地理对战争的影响

Step 4：历史文化推导
├── 历史纵深（100年以上）
├── 历史事件
├── 历史断层
├── 历史遗忘
└── 文化传承与断裂
```

**输出物**：
- 完整世界观设定文档
- 知识图谱实体录入

### 2.3 主题定位

**步骤**：

```
Step 1：确定核心主题
├── 选择永恒议题（权力、生存、牺牲、命运）
├── 确保超越个人层面
└── 准备不给明确答案

Step 2：设计多主题交织
├── 核心主题
├── 辅助主题
└── 主题层次关系

Step 3：检验主题
├── 是否触及永恒议题？
├── 是否超越个人？
├── 是否无简单答案？
└── 是否能承载史诗感？
```

## 三、第二阶段：大纲与结构

### 3.1 多线叙事设计

**步骤**：

```
Step 1：确定线索数量
├── 双线叙事（中篇）
├── 三线叙事（长篇）
├── 多线叙事（史诗级）
└── 群线叙事（史诗级巨著）

Step 2：设计线索独立性
├── 每条线索有独立主角
├── 每条线索有独立目标
├── 每条线索有独立冲突
└── 每条线索可单独成故事

Step 3：设计交织点
├── 信息交织点
├── 人物交织点
├── 事件交织点
└── 主题交织点

Step 4：设计汇聚结构
├── 预汇聚时机
├── 主汇聚时机
├── 终汇聚时机
└── 后汇聚结果
```

**输出物**：
- 多线叙事设计文档

### 3.2 人物小传创建

**步骤**：

```
Step 1：创建主角小传
├── 基础信息
├── 外貌描写
├── 性格与心理
├── 人物弧光设计
├── 独立命运线
├── 语言特征
└── 关系网络

Step 2：创建配角小传
├── 重要配角（独立命运线）
├── 辅助配角（功能角色）
└── 每人都有独立动机

Step 3：创建反派小传
├── 反派立场
├── 反派逻辑
├── 反派价值观
└── 道德灰色设计
```

**输出物**：
- 人物小传文档（见99-创作模板/人物小传模板.md）

### 3.3 冲突链条设计

**步骤**：

```
Step 1：设计主线冲突
├── 确定冲突类型
├── 设计三层矛盾
├── 设计冲突升级阶梯
└── 设计冲突结果与代价

Step 2：设计支线冲突
├── 与主线关联
├── 独立性设计
└── 汇聚点设计

Step 3：设计冲突链条
├── 前置冲突
├── 当前冲突
├── 后续冲突
└── 最终汇聚
```

**输出物**：
- 冲突设计文档（见99-创作模板/冲突设计模板.md）

### 3.4 章节大纲规划

**步骤**：

```
Step 1：确定卷结构
├── 每卷主题
├── 每卷冲突
├── 每卷高潮
└── 卷间关联

Step 2：规划章节
├── 每章目的
├── 每章场景
├── 每章悬念
└── 每章节奏

Step 3：设计黄金三章
├── 第一章：建立场景、引入主角、设置悬念
├── 第二章：展开冲突、深化人物、推进情节
├── 第三章：核心事件、情感投入、钩子设计
```

**输出物**：
- 章节大纲文档（见99-创作模板/章节大纲模板.md）

## 四、第三阶段：章节创作

### 4.1 场景设计

**创作前**：

```
Step 1：确定场景目的
├── 推进情节？
├── 塑造人物？
├── 传递信息？
└── 制造冲突？

Step 2：设计场景氛围
├── 核心氛围
├── 环境元素
├── 时间痕迹
└── 情感承载

Step 3：规划场景结构
├── 开场（引入）
├── 中段（展开）
└── 结尾（收束）
```

**创作中**：

```
Step 1：建立场景
├── 环境描写（具体而非抽象）
├── 氛围营造（克制表达）
└── 人物入场

Step 2：展开内容
├── 对话设计（有潜台词）
├── 冲突发展（有升级）
├── 信息传递（自然嵌入）
└── 情感投入（克制深沉）

Step 3：收束场景
├── 结果呈现
├── 悬念设置
└── 过渡下一场景
```

**输出物**：
- 场景设计文档（见99-创作模板/场景设计模板.md）

### 4.2 对话创作

**创作前**：

```
Step 1：确定对话功能分配
├── 推动情节 30%
├── 塑造角色 25%
├── 传递信息 15%
├── 制造冲突 15%
├── 营造氛围 10%
└── 揭示潜台词 5%

Step 2：设计角色语言特征
├── 句长
├── 用词
├── 语气
├── 话题倾向
└── 表达方式

Step 3：设计潜台词
├── 表面对话
├── 真实意图
└── 暗示方式
```

**创作中**：

```
Step 1：角色差异化
├── 遮住名字能否辨认是谁？
├── 语言是否符合角色性格？
└── 对话是否有潜台词？

Step 2：冲突升级
├── 对话推动冲突升级
├── 信息在冲突中传递
└── 情感在对话中深化

Step 3：格式规范
├── 避免"他说...他说...他说..."
├── 动作优于标签
├── 必要时省略标签
```

### 4.3 氛围营造

**创作原则**：

```
原则1：具体而非抽象
├── ❌ "无比悲伤"
├── ✅ "眼眶干涩，一滴泪都没有"

原则2：克制表达
├── ❌ "心中涌起巨大的痛苦"
├── ✅ 用动作、身体反应表达

原则3：情感承载于物象
├── ❌ "他非常思念她"
├── ✅ "他看着她留下的那只碗，碗沿还有她磕出的缺口"

原则4：避免渲染
├── 少用感叹号
├── 平静叙述重大事件
└── 让读者自行感受
```

### 4.4 悬念设计

**章内悬念**：

```
设计步骤：
├── 确定悬念类型（信息/结果/选择）
├── 设计悬念强度
├── 确定揭晓时机
└── 确保最终揭晓
```

**章末悬念（钩子）**：

```
设计步骤：
├── 确定悬念内容
├── 设计悬念强度
├── 确保读者期待
└── 与下一章关联
```

## 五、第四阶段：修订与精修

### 5.1 全文检查

**维度检查**：

```
□ 世界观检查
  ├── 历史纵深是否足够？
  ├── 内在逻辑是否一致？
  └── 世界观是否自生长？

□ 剧情检查
  ├── 命运驱动是否明确？
  ├── 历史回响是否设计？
  └── 结局是否有延续感？

□ 人物检查
  ├── 群像塑造是否完整？
  ├── 配角是否有独立命运线？
  └── 反派是否有立场和逻辑？

□ 冲突检查
  ├── 三层矛盾是否设计？
  ├── 冲突升级是否有阶梯？
  └── 胜利是否有代价？

□ 氛围检查
  ├── 厚重感是否营造？
  ├── 是否静默中的悲怆？
  └── 是否有时间沉淀感？

□ 叙事检查
  ├── 视角切换是否合理？
  ├── 悬念设计是否完整？
  └── 节奏控制是否得当？

□ 主题检查
  ├── 是否超越个人？
  ├── 是否触及永恒议题？
  └── 是否无简单答案？

□ 情感检查
  ├── 是否有集体情感？
  ├── 是否克制表达？
  └── 是否有情感代价？
```

### 5.2 技法对照

**使用检查清单**：

```
使用 01-创作检查清单.md：

Step 1：逐项检查
├── P0核心项：必须通过
├── P1重要项：尽量通过
└── P2精修项：优化通过

Step 2：计算评分
├── S：48+（史诗级标准）
├── A：40-47（优秀）
├── B：32-39（良好）
├── C：24-31（合格）
└── D：<24（需改进）

Step 3：针对性修正
├── 找出得分最低的维度
├── 对照技法文件修正
└── 重新检查评分
```

### 5.3 禁止项清理

**禁止项清单**：

```
□ AI味表达
  ├── "眼中闪过一丝XXX"
  ├── "心中涌起一股XXX"
  ├── "嘴角勾起一抹XXX"
  └── 全文搜索并替换

□ 抽象统计
  ├── "无数"、"成千上万"
  └── 替换为具体数字

□ 渲染式表达
  ├── 过多感叹号
  └── 删除或减少

□ 精确年龄
  ├── "XX岁的XXX"
  └── 改为人生阶段描述

□ Markdown格式
  ├── **XXX** 加粗
  └── 删除加粗
```

### 5.4 节奏调整

**节奏检查**：

```
Step 1：绘制节奏曲线
├── 标记每段落的节奏（快/中/慢）
├── 绘制整体曲线
└── 检查是否有起伏

Step 2：调整节奏
├── 连续快节奏段落 → 增加缓解段落
├── 连续慢节奏段落 → 增加紧张段落
├── 节奏突变 → 增加过渡段落
└── 节奏单调 → 设计节奏变化

Step 3：验证调整
├── 读者阅读体验是否舒适？
├── 张弛是否有规律？
└── 高潮是否有铺垫和缓解？
```

## 六、创作检查节点

### 6.1 阶段检查节点

| 阶段完成 | 检查内容 | 通过标准 |
|----------|----------|----------|
| **世界观完成** | 公理、势力、历史、力量体系 | 自洽且可推导 |
| **大纲完成** | 多线、人物、冲突、章节 | 结构完整 |
| **第一章完成** | 开篇技法、氛围、悬念 | 符合黄金三章 |
| **每章完成** | 场景、对话、悬念、节奏 | 符合章节模板 |
| **全卷完成** | 全文检查、技法对照 | 评分B以上 |
| **全文完成** | 全部维度检查 | 评分A以上 |

### 6.2 质量门禁

```
门禁1：世界观自洽
├── 公理从未被打破
├── 设定互相一致
└── 可自然推导

门禁2：人物真实
├── 配角有独立命运线
├── 反派有立场和逻辑
└── 道德灰色设计

门禁3：冲突有力
├── 三层矛盾设计
├── 升级阶梯完整
└── 有代价的胜利

门禁4：氛围厚重
├── 具体而非抽象
├── 克制表达
└── 史诗感

门禁5：禁止项为零
├── 无AI味表达
├── 无抽象统计
├── 无渲染式表达
└── 无精确年龄
```

## 八、实战应用清单

### 8.1 创作前准备

```
□ 阅读技法库核心文件
□ 确定创作主题和类型
□ 准备模板文件
□ 建立项目文件夹
□ 创建知识图谱
```

### 8.2 创作中跟进

```
□ 每完成一章，使用章节模板检查
□ 每完成一卷，使用检查清单评分
□ 发现问题，及时对照技法修正
□ 保持禁止项为零
□ 记录创作心得和技法验证
```

### 8.3 创作后验证

```
□ 全文检查清单评分
□ 技法应用对照
□ 禁止项最终清理
□ 节奏曲线验证
□ 主题深度检验
```
