## 快速索引

### 核心技法速查

| 维度 | 核心公式 | 关键技法 |
|-----|---------|---------|
| **世界观** | 历史纵深 + 逻辑一致 + 自生长 | 公理推导法、断层设计法、约束设计 |
| **剧情** | 命运驱动 + 历史回响 + 无简单结局 | 命运驱动设计、历史回响因果链、伏笔跨度 |
| **人物** | 群像命运 + 独立人格 + 道德灰色 | 群像塑造、独立命运线、选择代价 |
| **战斗** | 有代价胜利 + 群体牺牲 + 格局改变 | 代价设计、战术逻辑、群体牺牲描写 |
| **氛围** | 历史沉淀感 + 静默悲怆 + 物是人非 | 痕迹式描写、对比结构、克制表达 |
| **叙事** | POV切换 + 时间折叠 + 预言伏笔 | POV设计、时间折叠、史诗诗词 |
| **主题** | 超越个人 + 永恒议题 + 无答案 | 永恒议题设计、无答案困境 |
| **情感** | 集体情感 + 历史创伤 + 克制表达 | 集体情感设计、历史创伤、动作暗示 |
| **节奏** | 快慢交替 + 情绪曲线 + 时间感知 | 句子长度控制、时间折叠、情绪节奏 |

### 史诗级本质特征

> **史诗级著作 = 历史体量 + 群像命运 + 世界真实感 + 主题永恒性**

| 本质特征 | 说明 |
|----------|------|
| **历史体量** | 不是"今天的故事"，是"时代的故事" |
| **群像命运** | 不是"主角的故事"，是"多命运交织的故事" |
| **世界真实感** | 不是"设定表"，是"真实存在过的世界" |
| **主题永恒性** | 不是"个人议题"，是"人类/文明议题" |

## 检查清单使用

创作时使用 `01-创作检查清单.md`：

1. **创作前**：对照清单规划，确保设计符合标准
2. **创作中**：P0核心项持续对照，避免偏离
3. **创作后**：全量检查，计算评分
4. **修订时**：逐维度修正，提升评分

**评分标准**：
- S：48+（史诗级标准）
- A：40-47（优秀）
- B：32-39（良好）
- C：24-31（合格）
- D：<24（需改进）
