# 自然风景描写技法

通过日月星辰、山川河海、风霜雨雪等自然意象，营造氛围和意境。

---

## 一、日月星辰描写

### 1.1 太阳描写

**词汇库**：艳阳、旭日、朝阳、骄阳、斜阳、残阳、夕阳、火轮、金轮、绯红、朱红、殷红

**状态库**：
- **升起**：昔日东升、旭日当空、红日高照、火轮升腾、日上三竿
- **正午**：艳阳当空、日选中天、赤日炎炎、骄阳似火、烈日如焚
- **落下**：日落西山、红日西垂、红日西坠、夕阳西斜、落日余晖

**案例**：
- 头顶悬着一轮血色的月亮，这月亮不对劲，红得刺眼，光也是冷的，照在身上没有半点温度

### 1.2 月亮描写

**词汇库**：皓月、皎月、淡月、残月、晓月、冷月、玉盘

**状态库**：
- **圆满**：皓月当空、月满如盘、月华如水、月白风清、月圆花好
- **残缺**：残月如钩、月牙初上、月缺重圆
- **环境**：月上柳梢、月照东墙、月满西楼、月黑风高

**案例**：
- 月光惨淡，风清月明，风影婆娑

### 1.3 星辰描写

**词汇库**：寒星、星光、繁星、孤星、晨星、残星、群星、银河、星空

**状态库**：
- **密集**：群星灿烂、星星点点、繁星满天、繁星密布、星罗棋布
- **稀疏**：晨星寥落、星光朦胧、孤星闪烁
- **动态**：星垂平野、星光闪闪、星月交辉、星光璀璨、繁星入海

**流星**：彗星、耀眼、掠过、划破、陨落、转瞬即逝、一闪而过

## 二、山川河海描写

### 2.1 山岳描写

| 词汇 | 说明 | 案例 |
|------|------|------|
| **崇山峻岭** | 高大险峻的山岭 | 崇山峻岭，连绵起伏 |
| **千山万壑** | 山峦连绵，高低重叠 | 千山万壑，层峦叠嶂 |
| **山清水秀** | 风景优美 | 山清水秀，风景如画 |
| **湖光山色** | 有山有水的景色 | 湖光山色，美不胜收 |

**成语库**：巴山蜀水、白山黑水、锦绣河山、名山大川、仙山琼阁、重山复岭

### 2.2 水景描写

**河流**：
- 绿水青山、一马平川、楚水吴山

**海洋**：
- **状态**：海潮、海浪、涨潮、退潮、潮汐
- **动态**：惊涛拍岸、浪花汹涌、奔腾翻卷、汹涌澎湃、排山倒海
- **声音**：波涛万顷、声如金鼓、水花飞扬、疯狂海啸、水天相接

## 三、风霜雨雪描写

### 3.1 风描写

| 风类型 | 词汇 | 案例 |
|--------|------|------|
| **春风** | 和风、清风、暖风、习习、徐徐 | 春风拂面，春风宜人 |
| **夏风** | 疾风、热风、阵阵 | 南风徐徐，东风送暖 |
| **秋风** | 朔风、凄风、瑟瑟 | 秋风飒飒，冷风飕飕 |
| **冬风** | 寒风、北风、凛冽 | 北风呼啸，寒风刺骨 |

**成语**：风平浪静、风调雨顺、风和日丽、风卷残云、风起云涌

### 3.2 霜雪描写

**霜**：
- 玉霜、银霜、晚霜、霜天、霜花、白霜、霜冻

**雪**：
- **状态**：白雪、瑞雪、残雪、鹅毛雪
- **形态**：毛茸茸、亮晶晶、白茫茫
- **动态**：大雪纷飞、万里飘雪、雪飘如絮、漫天大雪、白雪皑皑
- **覆盖**：银装铺地、银装素裹、粉妆玉砌、玉树琼枝

### 3.3 雨描写

| 雨类型 | 词汇 | 案例 |
|--------|------|------|
| **小雨** | 微雨、烟雨、绵绵、蒙蒙、毛毛雨 | 细雨如烟，春雨无声 |
| **中雨** | 疾雨、疏疏雨、雨纷纷、雨霏霏 | 阴雨连绵，寒风细雨 |
| **大雨** | 骤雨、瓢泼、雨茫茫 | 大雨倾盆，狂风暴雨 |

### 3.4 雾露描写

**雾**：
- 浓雾、密雾、重雾、薄雾、迷雾、雾霭
- 晨雾笼罩、云雾升腾、雾气缭绕、雾锁云笼、雾似轻纱

**露**：
- 露水、露珠、朝露、寒露、夜露、晶莹
- 白露沾草、露水滋润、露珠闪烁、露珠晶莹、晶莹剔透

## 八、快速调用指南

### 8.1 风景描写顺序

```
1. 大场景（天空/远山）
2. 中景（建筑/树木）
3. 近景（人物/细节）
4. 动态元素（风/云/水流）
5. 感官细节（声音/气味/触感）
```

### 8.2 氛围营造技巧

| 氛围类型 | 推荐意象 | 案例 |
|---------|---------|------|
| **悲壮** | 血月、残阳、荒原、枯树 | 头顶悬着一轮血色的月亮，红得刺眼 |
| **宁静** | 皓月、清霜、微露、轻风 | 皓月当空，风清月朗 |
| **压抑** | 乌云、浓雾、暴雨、沉雷 | 乌云密布，重云如铅 |
| **希望** | 旭日、春风、新芽、彩虹 | 旭日当空，大地回春 |

### 8.3 注意事项

| ❌ 禁止 | ✅ 替代 |
|--------|--------|
| 堆砌辞藻无目的 | 为情节/情感服务 |
| 脱离故事背景 | 与故事氛围统一 |
| 静态描写过多 | 加入动态元素 |
| 忽略感官细节 | 调动视觉/听觉/嗅觉/触觉 |
