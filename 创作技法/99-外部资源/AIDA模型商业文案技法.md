# AIDA模型商业文案技法

> 来源：营销学经典AIDA模型
> 整理时间：2026-03-31

---

## 一、理论来源

AIDA模型是西方推销学经典模型，核心命题：

> **Attention（注意）→ Interest（兴趣）→ Desire（欲望）→ Action（行动）**

这一模型可直接转化为小说开篇设计、章节钩子设计、读者转化等创作技法。

---

## 二、AIDA模型详解

### 2.1 四阶段总览

| 阶段 | 英文 | 定义 | 小说转化 |
|------|------|------|----------|
| **A** | Attention | 引起注意 | 开篇钩子、标题设计、悬念设置 |
| **I** | Interest | 激发兴趣 | 情节展开、角色塑造、世界观呈现 |
| **D** | Desire | 诱发欲望 | 情感共鸣、目标认同、代入感建立 |
| **A** | Action | 促使行动 | 章节结尾钩子、追更动力、情感投入 |

### 2.2 Attention（注意）

**目标**：让读者停下脚步，开始阅读。

**设计公式**：
```
注意力 = 冲突强度 × 新奇程度 × 相关性

冲突强度：打破读者的期待
新奇程度：展示读者未见过的东西
相关性：与读者的需求/兴趣相关
```

**小说应用**：
```
开篇注意力设计：

1. 冲突开篇：
"血色染红了雪地，他低头看着手中的剑。"
——直接展示冲突画面

2. 悬念开篇：
"那天之后，所有人都死了，除了我。"
——引发读者好奇

3. 反差开篇：
"他是最年轻的宗主，也是最大的废物。"
——打破读者期待

4. 新奇开篇：
"在修仙界，有一种人叫'道奴'，他们的命不属于自己。"
——展示新颖世界观

5. 提问开篇：
"你有没有想过，为什么所有穿越者都会去考科举？"
——引发读者思考
```

**注意力公式**：
```
好开篇 = 1句话钩子 + 1个悬念问题 + 1个冲突画面

标题设计：
吸睛标题 = 数字/对比 + 痛点/利益 + 独特性

示例：
"七岁登基，他用了十年把烂摊子变成帝国"
"修仙界最弱宗门？三百年后他们统治了世界"
```

### 2.3 Interest（兴趣）

**目标**：让读者持续阅读，保持兴趣。

**设计公式**：
```
兴趣 = 信息价值 × 情感连接 × 好奇心

信息价值：读者能获得什么
情感连接：与角色的共鸣
好奇心：想知道接下来发生什么
```

**小说应用**：
```
兴趣维持设计：

1. 信息分层释放：
├── 开篇：建立基础认知
├── 前三章：核心世界观
├── 中段：逐步深化
└── 后段：完整揭示

2. 角色兴趣点：
├── 独特的性格特征
├── 令人好奇的背景
├── 明确的目标动机
└── 成长的潜力

3. 世界观兴趣点：
├── 独特的设定体系
├── 有趣的社会结构
├── 神秘的历史背景
└── 暗示的更大世界

4. 情节兴趣点：
├── 悬念设置
├── 反转设计
├── 冲突升级
└── 节奏变化
```

**兴趣维持检查清单**：
```
每章兴趣检查：
□ 是否有新信息？
□ 是否有新冲突？
□ 是否推进剧情？
□ 是否深化角色？
□ 是否设置钩子？
```

### 2.4 Desire（欲望）

**目标**：让读者产生强烈的阅读欲望和代入感。

**设计公式**：
```
欲望 = 需求识别 × 情感共鸣 × 代入感建立

需求识别：识别读者的心理需求
情感共鸣：触动读者的情感
代入感建立：让读者成为角色
```

**读者心理需求分类**：

| 需求类型 | 描述 | 满足方式 |
|----------|------|----------|
| **成长需求** | 看到主角变强 | 升级、突破、成长 |
| **情感需求** | 情感共鸣体验 | 爱情、友情、亲情 |
| **正义需求** | 看到善有善报 | 打脸、复仇、伸张正义 |
| **逃避需求** | 逃离现实压力 | 异世界、奇幻设定 |
| **归属需求** | 找到认同感 | 群体、门派、组织 |
| **成就需求** | 获得成就感 | 主角成功、读者满足 |

**小说应用**：
```
欲望触发设计：

1. 目标认同：
├── 给主角一个合理的目标
├── 让读者希望主角成功
├── 设置障碍增加难度
└── 最终成功满足欲望

2. 情感共鸣：
├── 角色经历读者能理解
├── 情感表达真实可信
├── 道德困境引发思考
└── 成功/失败都能共鸣

3. 代入感建立：
├── 角色有读者认可的特质
├── 经历读者能感同身受
├── 选择读者会做出类似判断
└── 成功读者感到满足

4. 欲望升级：
├── 从小欲望到大欲望
├── 从个人到群体
├── 从物质到精神
└── 从短期到长期
```

### 2.5 Action（行动）

**目标**：让读者采取行动——继续阅读、追更、推荐。

**设计公式**：
```
行动 = 紧迫感 × 简单性 × 回报预期

紧迫感：必须现在行动
简单性：行动很简单
回报预期：行动后有回报
```

**小说应用**：
```
行动触发设计：

1. 章节结尾钩子：
├── 悬念式："当他推开门的那一刻，所有人都在看着他。"
├── 危机式："敌人已经包围了整个山门。"
├── 反转式："原来她一直都在骗他。"
├── 新信息式："他终于知道了真相。"
└── 选择式："他必须在今天做出选择。"

2. 追更动力：
├── 未解决的悬念
├── 即将发生的大事
├── 角色的命运未知
└── 高潮即将来临

3. 推荐动力：
├── 惊艳的开篇
├── 独特的设定
├── 精彩的高潮
└── 深刻的主题

4. 情感投入：
├── 角色命运牵动读者
├── 故事走向引发讨论
├── 世界观值得深入探索
└── 主题引发思考
```

## 三、AIDA在小说创作中的完整应用

### 3.1 开篇AIDA设计

```
开篇AIDA结构：

第一章：Attention
├── 钩子：1句话悬念
├── 冲突：打破平衡
├── 新奇：展示独特世界观
└── 引入主角

第二章：Interest
├── 主角性格展示
├── 世界观展开
├── 核心冲突提出
└── 配角引入

第三章：Desire
├── 主角目标确立
├── 障碍出现
├── 代入感建立
└── 章节钩子设置

后续章节：持续Interest + Desire + Action循环
```

### 3.2 章节AIDA设计

```
章节AIDA结构：

开头（Attention）：
├── 承接上章钩子
├── 快速进入场景
├── 新冲突或新信息
└── 引起阅读兴趣

中段（Interest + Desire）：
├── 情节推进
├── 角色深化
├── 冲突升级
└── 代入感增强

结尾（Action）：
├── 设置新钩子
├── 提出新悬念
├── 制造追更动力
└── 情感满足或期待
```

### 3.3 场景AIDA设计

```
场景AIDA结构：

入场（Attention）：
├── 场景快速建立
├── 角色明确
├── 冲突/目标清晰
└── 引起注意

展开（Interest）：
├── 细节展开
├── 对话推进
├── 冲突升级
└── 保持兴趣

高潮（Desire）：
├── 情感饱和
├── 关键选择
├── 最大张力
└── 激发欲望

收尾（Action）：
├── 结果呈现
├── 情感释放
├── 过渡下一场景
└── 推动行动
```

## 五、标题与简介AIDA设计

### 5.1 标题设计公式

```
吸睛标题公式：

公式1：数字 + 对比 + 结果
"七年时间，他把一个破落宗门变成了修仙界第一"

公式2：身份 + 反差 + 结果
"她是废材小姐，却成了三界最强的炼丹师"

公式3：悬念 + 好奇
"那个消失十年的人回来了，整个修仙界都在颤抖"

公式4：目标 + 障碍 + 反转
"我只是想当个普通人，为什么所有人都来杀我？"

公式5：独特 + 新奇
"重生后，她决定把所有害过她的人...都变成徒弟"
```

### 5.2 简介设计公式

```
简介AIDA结构：

钩子（Attention）：
├── 1-2句话悬念
├── 核心冲突
├── 独特设定
└── 吸引注意

展开（Interest）：
├── 主角介绍
├── 世界观概述
├── 核心矛盾
└── 激发兴趣

动机（Desire）：
├── 主角目标
├── 面临障碍
├── 成长可能
└── 诱发欲望

召唤（Action）：
├── 故事走向暗示
├── 核心看点总结
├── 追读动力
└── 促使行动
```

**简介模板**：
```
【钩子】一句话悬念
【展开】主角是谁 + 世界观是什么 + 核心冲突
【动机】主角的目标 + 面临的障碍
【召唤】这个故事将如何精彩
```

## 六、实战应用清单

### 6.1 创作前：AIDA规划

```
□ 开篇钩子：能一句话吸引注意吗？
□ 兴趣维持：前三章有足够的信息量和冲突吗？
□ 欲望触发：读者会希望主角成功吗？
□ 行动驱动：章节结尾有追读动力吗？
□ 标题简介：符合AIDA结构吗？
```

### 6.2 创作中：AIDA检查

```
每章检查：
□ 开头是否有钩子？
□ 中段是否保持兴趣？
□ 情感是否有共鸣？
□ 结尾是否有追读动力？

场景检查：
□ 是否有注意力点？
□ 是否有信息价值？
□ 是否有情感共鸣？
□ 是否推动阅读行动？
```

### 6.3 修订时：AIDA审查

| 检查项 | 标准 |
|--------|------|
| 开篇吸引力 | 能否在前100字抓住读者 |
| 兴趣持续性 | 每章是否有新信息/冲突 |
| 代入感强度 | 读者是否关心主角命运 |
| 追读动力 | 章节结尾是否有钩子 |

## 七、核心公式总结

```
AIDA = Attention（注意）→ Interest（兴趣）→ Desire（欲望）→ Action（行动）

注意力 = 冲突强度 × 新奇程度 × 相关性

兴趣 = 信息价值 × 情感连接 × 好奇心

欲望 = 需求识别 × 情感共鸣 × 代入感建立

行动 = 紧迫感 × 简单性 × 回报预期

好开篇 = 钩子 + 悬念 + 冲突

追读动力 = 未解悬念 + 角色命运 + 高潮期待
```
