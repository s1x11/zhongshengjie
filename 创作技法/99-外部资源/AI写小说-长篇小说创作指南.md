# AI写小说：长篇小说创作指南

使用大模型生成长篇小说的保姆级教程，涵盖从灵感到完本的全流程。

---

## 一、对齐认知（第一步）

在开始正式创作之前，先让AI和你对"长篇小说"有共同的认识。

### 长篇节奏与结构

相较于短篇故事，长篇小说：
- 包含多个情节单元（卷/章节）
- 张弛有度：既要有小高潮不断吸引读者，也要有整体起承转合推动主线发展
- 需要在每章结尾设置悬念以吸引读者追更

**提示词范例**：
```
你是一名资深网络小说作者。我准备写一部长篇都市小说连载，希望了解在创作长篇时需要注意哪些问题，以及都市题材读者喜欢的元素有哪些？请结合长篇小说的结构节奏和网络平台读者偏好给出建议。
```

### 目标平台与读者偏好

不同平台有不同的受众偏好：

| 平台 | 特点 |
|------|------|
| 起点 | 男频为主，慢热、世界观复杂 |
| 晋江 | 女频为主，情感细腻、人设重要 |
| 番茄 | 快节奏、强冲突、爽文标签点击率高 |

研究显示，三四线城市用户对"逆袭打脸""直播致富"等爽文标签点击率比一二线城市用户高出**32%**。

### 技术准备

- 使用高性能版本（如GPT-4）
- 保持对话上下文完整
- 付费版本具有更大的上下文窗口，能在长对话中记忆更多内容

---

## 二、找灵感，确定主题（第二步）

选定小说的大体类型和核心主题，激发出一个有吸引力的故事想法。

### 灵感来源

| 来源 | 说明 |
|------|------|
| 热点话题 | 社会热点、影视IP |
| 经典套路创新 | 反套路、新视角 |
| 个人经历 | 真实情感更容易打动读者 |
| AI辅助 | 让AI生成多个创意方向 |

### 主题确定

好的主题应该是：
- 有情感共鸣
- 有探讨空间
- 有读者基础

---

## 三、世界观设定（第三步）

### 世界观生成公式

```
请生成[题材]世界观，要求包含：
- 3个特色种族（如半人鱼+机械师+灵蝶族）
- 2个对立势力（如光明教廷VS暗影议会）
- 独特的力量体系规则
- 核心冲突来源
```

### 世界观要素

| 要素 | 说明 |
|------|------|
| 地理设定 | 地图、气候、资源分布 |
| 社会结构 | 阶层、组织、势力 |
| 力量体系 | 等级、规则、代价 |
| 历史背景 | 大事件、传说、伏笔 |

---

## 四、人物设定（第四步）

### 完整人物卡模板

```
【角色完整档案】
姓名：
年龄：
外貌（150字以内，极具画面感）：
性格核心三要素（三个形容词+一句话解释）：
最深层的恐惧/创伤/执念：
最渴望得到的东西（表面愿望 vs 内心真正渴望）：
三观（对金钱/爱情/权力/亲情/死亡的态度）：
特殊习惯/小动作/口头禅（3～5个）：
擅长领域/技能树（3～5项）：
最弱的领域/致命弱点（2～3项）：
```

### 人物关系网

- 主角：故事核心，需要有成长弧线
- 配角：推动剧情，各司其职
- 反派：制造冲突，有合理动机

---

## 五、大纲设计（第五步）

### 大纲结构

```
第一卷：开篇（10-15%）
- 世界观引入
- 主角出场
- 核心冲突埋设

第二卷：发展（25-30%）
- 冲突升级
- 人物关系深化
- 新势力/新地图

第三卷：高潮（30-35%）
- 最大冲突爆发
- 关键转折
- 主角成长

第四卷：收尾（20-25%）
- 线索收束
- 情感升华
- 结局/伏笔
```

### 章节设计

每章应包含：
1. 主要内容（一句话概括）
2. 冲突点
3. 情绪点
4. 伏笔/悬念

---

## 六、持续写作（第六步）

### 章节推进模板

```
当前进度：第X章，共Z万字
当前情节点：[一句话总结]

本次写作要求：
1. 字数：1500-2500字
2. 必须包含：冲突升级/新信息揭露/情感转折/悬念埋设（至少三项）
3. 节奏控制：30%铺垫 + 40%冲突高潮 + 30%收尾悬念
4. 结尾必须有强力钩子

情节方向参考：
[粘贴大纲核心节点]
```

### 保持一致性

每写3-5章用一次检查器：
```
请对比之前的所有人物卡和已写章节，检查是否存在：
- OOC（角色行为不一致）
- 设定冲突
- 世界观bug
- 时间线错误
```

## 七、常见问题解决

### 卡文怎么办？

1. 回顾大纲，确认当前位置
2. 分析角色心理状态
3. 让AI给出3种可能的发展方向
4. 选择最合适的继续

### 逻辑崩了怎么办？

1. 暂停写作
2. 整理所有设定和伏笔
3. 找出矛盾点
4. 设计修复方案（可以是新伏笔）

### 节奏失控怎么办？

1. 检查每章字数是否合理
2. 确认冲突密度
3. 调整章节划分
4. 必要时合并或拆分章节

## 八、长篇小说检查清单

### 开篇检查

- [ ] 前500字是否有钩子？
- [ ] 第一章是否建立了核心冲突？
- [ ] 主角是否出场并有明确目标？
- [ ] 世界观是否自然引入？

### 中段检查

- [ ] 每3-5章是否有小高潮？
- [ ] 人物关系是否有发展？
- [ ] 是否有新冲突引入？
- [ ] 伏笔是否有埋设和回收？

### 收尾检查

- [ ] 主要冲突是否解决？
- [ ] 人物弧线是否完成？
- [ ] 伏笔是否回收？
- [ ] 结局是否令人满意？
