# AI写小说：完整Prompt模板体系

专为想要用大模型（Claude / GPT-4o / 通义千问 / DeepSeek / Gemini 等）进行高质量小说创作准备，按创作流程最常使用的顺序排列。

---

## 一、人物设定（Character Design）

> **核心原则**：越详细的人物卡 → 模型越能保持角色一致性（尤其是长篇、需要几十万字以上的作品）

### 完整人物卡模板

```
你现在要创作一本[类型，如：都市悬疑/仙侠/甜宠/克苏鲁/无限流]小说。

请严格按照以下人物卡扮演角色，任何时候都不能OOC（Out of Character）。

【角色完整档案】
姓名：
年龄：
外貌（150字以内，极具画面感）：
身高/体型/标志性特征：
声音特点（语速、音色、口癖）：
性格核心三要素（用三个形容词+一句话解释）：
MBTI / 九型人格 / 星座（可选，但有助于模型理解）
最深层的恐惧/创伤/执念：
最渴望得到的东西（表面愿望 vs 内心真正渴望）：
三观（对金钱/爱情/权力/亲情/死亡的态度）：
特殊习惯/小动作/口头禅（3～5个，越具体越好）：
与主角的关系（当前阶段）：
当前最强烈的内心冲突/矛盾：
擅长领域/技能树（列出3～5项）
最弱的领域/致命弱点（列出2～3项）

【语言风格要求】
1. 永远使用第一人称内心独白 + 第三人称叙述结合
2. 对话必须符合人物年龄、身份、教育背景、地域特征
3. 禁止使用过于现代、网络化的梗，除非角色设定本身就是网文作者/键盘侠类型
4. 情绪强烈时允许出现断句、语气词、省略号、感叹号叠加

【当前场景】
时间：202X年X月X日 凌晨3:17
地点：雨夜中的老旧出租屋/千年古刹/星际战舰指挥舱/……
当前情绪基调：压抑/狂热/甜蜜到发苦/濒临崩溃/……
刚刚发生的事情：（一句话概括触发事件）

现在请以这个角色的视角，写一段300～500字的内心独白+环境描写+少量对话（如果有其他人出现）
```

### 小技巧

把人物卡保存为独立的「知识库文件」或「Claude Project」，每次创作都先加载人物卡，能极大提高角色一致性。

## 二、情节推进（Plot Development）

> **最容易崩的环节**

### 常用情节推进Prompt模板

```
当前小说进度：
已完成章节：第X章～第Y章，共Z万字
当前最新情节点：（一句话总结上一次结束时的状态）

写作要求：
1. 本次推进约1500～2500字（可调整）
2. 必须包含：冲突升级/新信息揭露/情感转折/悬念埋设 中的至少三项
3. 节奏控制：前30%铺垫+情绪酝酿，中40%冲突爆发，高潮，后30%收尾+新悬念
4. 保持全书基调：[暗黑/治愈/爽文/克制/群像/单元剧等]
5. 每个主要角色都要有「态度变化」或「行为决策」的刻画（哪怕很微小）
6. 结尾必须留下至少一个强力钩子（hook），让读者想看下一章

情节大方向参考（请严格遵守，不要自己魔改主线）：
[在这里粘贴你之前写好的故事大纲或核心情节节点]

现在请继续写作下一章，章节标题要足够吸引人。
```

### 进阶变体

| 需求 | 指令 |
|------|------|
| 只推进一个小事件 | 「只写XX与YY第一次正式对峙的完整过程」 |
| 写转折点 | 「重点描写主角发现真相那一刻的心理崩塌」 |
| 写高潮 | 「按照三幕式结构写最终决战」 |

## 三、对话写作（Dialogue）

> **最能拉开差距的部分**

### 顶级对话Prompt模板

```
你现在要写一段高质量小说对话，场景是：[具体场景描述]

参与对话角色（严格按照人物卡）：
1. A：[姓名] - [一句话性格气质总结]
2. B：[姓名] - [一句话性格气质总结]
（可增加更多角色）

对话要求：
1. 每句话都要符合这个人物的身份、年龄、教育程度、地域、当前情绪状态
2. 真实人类对话特点：断句、抢话、口吃、转移话题、说一半留一半、答非所问
3. 避免「说明式对话」（禁止"你是说……吗？"这种低级解释）
4. 每个人物要有独特的语言节奏和标志性表达方式
5. 对话占比约60～70%，其余为动作、神态、环境、心理活动穿插
6. 对话总长度控制在[300～800]字之间

当前情境与目的：
- 刚刚发生了：[一句话概括前情]
- 这段对话的核心目的是：[逼问真相/试探底线/表白/宣战/和解/互相伤害/调情等]
- 预期情绪走向：[由冷到热/逐渐失控/暗流涌动/表面和谐内心撕裂等]

现在请开始写作这段对话。
```

### 快速提升对话质感的5个小技巧

1. **加入地域方言特征**，但不要过度
2. **不同角色使用不同的标点习惯**（！？…省略号使用频率不同）
3. **允许角色说脏话/黑话/土味情话**，前提是符合人设
4. **大量使用动作打断对话**：他冷笑一声/她突然顿住/手指攥紧杯子
5. **对话中埋设信息差和误会**（最容易制造张力）

## 四、长篇小说保命三件套

### 一致性检查器（每写3～5章用一次）

```
请严格对比我之前的所有人物卡和已写章节，检查下面这段文字是否存在任何OOC、设定冲突、世界观bug、时间线错误。
如果有，请逐条列出并给出修改建议。
如果没有，请直接回复「全线一致，无需修改」。
```

### 章节衔接器（写完一章想接下一章时用）

```
上一章结尾最后三句话是：
[粘贴]

请分析当前所有主要角色的心理状态、所处位置、掌握的信息差。
给出接下来最合理、最有张力的三种可能发展方向（每种一句话）。
我选其中一个，你再开始写下一章。
```

### 全书节奏把控器（写到1/3、1/2、2/3处用）

```
当前已完成字数：[X万字]，预计全书目标：[Y万字]
已出现的主要冲突点：[列出3～5个]
当前节奏感受：过快/过慢/比较合适

请给出接下来20～30%的剧情节奏建议，包括：
1. 需要解决的旧冲突
2. 需要引入的新冲突/反派/世界观层级提升
3. 每个主要角色预期完成的成长弧光阶段
4. 建议的高潮节点位置
```

## 五、核心指令模板（爆款向）

### 烂梗剔除+热点融合

```
你现在是阅书无数的网文主编，深耕[古言/科幻/悬疑]赛道5年，爆款率80%以上。

请审视我这个核心创意：[比如：穿越成冷宫弃妃，靠医术逆袭]

帮我找出市面上已经写烂的3个梗并删掉；

再结合2025年该赛道热门元素（比如古言的"轻权谋""中医食疗"，科幻的"意识上传伦理"），

给我3个最反直觉、最吸睛的开篇方向，每个方向必须带1个核心冲突+1个情绪钩子，语言风格要贴合[番茄/晋江/起点]平台调性。
```

### 反套路设定生成

```
作为专业小说家，为我的[奇幻/悬疑/仙侠]题材小说生成3个独特设定组合，要求包含：

1个反套路世界观元素（别搞"灵气复苏""系统绑定"这种烂大街的，参考《诡秘之主》的序列晋升体系）；

1个具有道德困境的主角身份（比如"检察官被迫为黑帮辩护""神医只能靠杀人续命"）；

1个看似微小但影响全局的关键物品（比如"会记录谎言的梳子""能交换记忆的旧邮票"），

每个设定组合配50字左右的创意解读，说明为什么吸睛。
```

## 六、都市爽文分类与适配

### 第一类：都市生活（现实向）

| 维度 | 说明 |
|------|------|
| 核心特点 | 完全基于现实世界，无金手指 |
| 典型题材 | 官场、商战、纯校园、都市言情 |
| 平台分布 | 起点、晋江（偏写实感、慢节奏） |
| 适合人群 | 有社会阅历的创作者 |

### 第二类：都市超现实（金手指）

| 维度 | 说明 |
|------|------|
| 核心特点 | 世界遵循现实逻辑，主角有金手指 |
| 典型题材 | 重生、异能、轻系统流 |
| 平台分布 | 起点、晋江、番茄 |
| 适合人群 | **新人首选！** |

**为什么对新人最友好？**
- 金手指降低创作难度
- 爽点好做且接地气
- 容错率高
