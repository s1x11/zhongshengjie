# 认知心理学思维描写技法

> 来源：认知心理学核心理论、Daniel Kahneman《思考，快与慢》
> 整理时间：2026-03-31

---

## 一、理论基础

### 1.1 信息加工理论

```
环境刺激 → 感觉记忆 → 注意选择 → 短时/工作记忆 → 编码 → 长时记忆 → 提取 → 行为反应
```

### 1.2 三阶段记忆模型

| 记忆类型 | 保持时间 | 容量 | 小说应用 |
|----------|----------|------|----------|
| 感觉记忆 | <1秒 | 较大 | 描写瞬间印象、闪回片段 |
| 工作记忆 | 15-30秒 | 4-7个组块 | 展现角色即时思考、推理 |
| 长时记忆 | 永久 | 无限 | 角色背景故事、创伤记忆 |

### 1.3 双系统理论

| 系统 | 特点 | 小说应用 |
|------|------|----------|
| **系统1（快思考）** | 快速、自动、直觉、无意识 | 危机时刻本能反应 |
| **系统2（慢思考）** | 缓慢、受控、分析、有意识 | 深思熟虑场景 |

## 二、认知偏差类型

### 2.1 决策偏差精选

| 偏差 | 定义 | 小说应用 |
|------|------|----------|
| **锚定效应** | 过度依赖第一个信息 | 谈判、初次见面 |
| **确认偏差** | 偏好符合信念的信息 | 固执己见、自我欺骗 |
| **后见之明偏误** | 事后认为更易预测 | 事后诸葛亮 |
| **过度自信效应** | 高估自身能力 | 盲目自信导致失败 |
| **沉没成本谬误** | 因已投入而继续错误决策 | 无法放手、执念 |
| **损失趋避** | 对损失敏感度高于收益 | 害怕改变 |
| **框架效应** | 同一信息不同呈现导致不同决策 | 说服场景 |

### 2.2 社会认知偏差

| 偏差 | 定义 | 小说创作应用 |
|------|------|--------------|
| 基本归因谬误 | 高估个人特质、低估情境因素 | 角色误解他人动机 |
| 晕轮效应 | 单一正面特征泛化为整体优秀 | 爱情中的盲目 |
| 刻板印象 | 基于类别判断个体 | 偏见主题 |

### 2.3 记忆错误

| 偏差 | 定义 | 小说应用 |
|------|------|----------|
| 误导性信息效应 | 事后信息扭曲原始记忆 | 目击证词不可靠 |
| 闪光灯记忆 | 情绪激动事件的生动但不准确记忆 | 创伤场景 |
| 峰终法则 | 记忆由峰值和结束时刻决定 | 经历回顾 |

## 三、图式理论

### 3.1 图式概念

**图式**：组织知识的心理框架/结构，帮助快速理解和预测。

| 类型 | 描述 | 小说应用 |
|------|------|----------|
| 人物图式 | 对某类人的认知框架 | 角色设定、期望违背 |
| 事件图式/脚本 | 对常规事件序列的预期 | 场景设计、打破常规 |
| 自我图式 | 关于自己的认知结构 | 身份认同、成长转变 |

### 3.2 图式运作机制

```
同化（用现有图式解释新经验）
      ↓
  认知冲突
      ↓
顺应（修改图式以适应新经验）
```

**小说应用**：角色成长弧线 = 图式的顺应过程

## 四、思维模式类型

| 思维模式 | 特点 | 适用于角色 |
|----------|------|------------|
| 全或无思维 | "要么完美，要么彻底失败" | 完美主义者 |
| 灾难化思维 | "如果出错就全完了" | 焦虑症角色 |
| 过度概括 | "一次失败 = 永远失败" | 抑郁倾向 |
| 情绪推理 | "我感觉自己是废物，所以我就是" | 低自尊角色 |
| 应该陈述 | "我应该/必须..." | 高压力角色 |

## 五、人物思维描写技法

### 5.1 工作记忆限制描写

> 她试图同时记住密码、车牌号和刚才的对话，大脑像过载的电脑一样卡住了。

### 5.2 认知偏差外显化

> 他只看到了支持自己观点的证据，其他一切都成了背景噪音。

### 5.3 图式冲突展现

> 在她的世界里，好人不会说谎——可他说这话时眼神那样真诚。她的大脑拒绝处理这个矛盾。

### 5.4 情绪对认知的扭曲

> 愤怒的时候，他的世界只剩下非黑即白；冷静下来后，灰色地带才重新浮现。

## 六、角色成长弧线设计

```
初始状态：固化的自我图式/世界图式
    ↓
挑战事件：与图式冲突的经验
    ↓
认知失调：痛苦的心理张力
    ↓
抵抗阶段：用各种偏差维护旧图式
    ↓
转折点：无法否认的证据/体验
    ↓
顺应过程：痛苦的图式重构
    ↓
整合完成：新图式形成，行为改变
```

## 七、核心公式

```
认知升级 = 实践积累 + 挫折反思 + 顿悟飞跃

思维质量 = 认知深度 × 元认知能力 × 偏差觉察

角色成长 = 图式冲突强度 × 顺应难度 × 整合程度
```
