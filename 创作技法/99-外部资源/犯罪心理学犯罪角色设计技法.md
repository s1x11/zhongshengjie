## 一、犯罪动机类型

### 1.1 中国刑法学11类动机

| 动机类型 | 核心驱动力 | 角色设计要点 |
|----------|------------|--------------|
| 政治动机 | 反社会需要 | 极端意识形态 |
| 财物动机 | 物质需求 | 贫困/贪婪驱动 |
| 性动机 | 性本能满足 | 性心理扭曲 |
| 报复动机 | 复仇心理 | 受伤害记忆、长期积怨 |
| 嫉妒动机 | 攀比心理 | 自卑转攻击 |
| 好奇动机 | 探索欲望 | 病态好奇心 |
| 友谊动机 | 义气/哥们情 | 忠诚错位 |
| 恐惧动机 | 自保心理 | 被逼入绝境 |
| 虚荣动机 | 面子需求 | 自恋型人格 |
| 信仰动机 | 宗教/理念 | 极端信念 |
| 集合动机 | 多种动机交织 | 多层次心理 |

### 1.2 工具性犯罪 vs 表达性犯罪

| 类型 | 特征 | 典型案例 |
|------|------|----------|
| 工具性犯罪 | 冷静、有计划、功利 | 抢劫、预谋杀人 |
| 表达性犯罪 | 冲动、激情、失控 | 激情杀人、酒后暴力 |

## 二、犯罪心理画像技术

### 2.1 FBI犯罪分类

| 特征维度 | 有组织型罪犯 | 无组织型罪犯 |
|----------|--------------|--------------|
| 智商 | 中上水平 | 平均水平以下 |
| 社交能力 | 良好，有魅力 | 差，孤立 |
| 犯罪计划 | 精心策划 | 临时起意 |
| 受害者选择 | 特定类型 | 随机 |
| 犯罪现场 | 整洁 | 混乱 |
| 心理状态 | 反社会人格 | 精神分裂 |

### 2.2 侧写要素

```
□ 年龄范围（基于犯罪成熟度）
□ 性别（统计概率）
□ 婚姻状况
□ 职业/教育水平
□ 居住环境
□ 心理疾病史
□ 前科记录
□ 与警方关系
```

## 三、连环杀手心理特征

### 3.1 核心心理障碍

| 心理特征 | 表现 | 小说描写要点 |
|----------|------|--------------|
| 精神变态 | 缺乏同理心、悔意 | 表面魅力，内心冰冷 |
| 自恋型人格 | 夸大自我重要性 | 认为自己"特殊" |
| 施虐倾向 | 从他人痛苦中获得快感 | 折磨过程描写 |

### 3.2 典型成长背景

```
童年创伤（高频特征）：
├── 身体虐待（50%+案件）
├── 性虐待
├── 精神虐待/忽视
├── 早年丧父（半数案件）
└── 与母亲关系扭曲

行为发展轨迹：
童年虐待 → 心理创伤 → 精神障碍 → 暴力行为升级 → 连环犯罪
```

### 3.3 早期预警信号

- 虐待动物
- 尿床（超过正常年龄）
- 纵火
- 社交孤立
- 对死亡/暴力迷恋

## 四、犯罪角色设计框架

### 4.1 角色档案模板

```
角色档案 = {
  基本信息：{年龄，性别，职业，教育，家庭背景},
  心理特征：{人格类型，心理障碍，智商},
  动机系统：{核心动机，次要动机，动机冲突},
  成长轨迹：{童年经历，关键转折点，心理变化},
  行为模式：{作案手法，签名特征，冷却期},
  社会网络：{家庭关系，朋友圈，孤立程度},
  认知模式：{自我合理化逻辑，世界观}
}
```

### 4.2 犯罪心理形成模式

| 模式 | 特征 | 适用角色类型 |
|------|------|--------------|
| 渐变式 | 从量变到质变的渐进过程 | 职业罪犯、黑化角色 |
| 突变式 | 突发刺激导致迅速犯罪 | 激情犯罪 |
| 机遇式 | 偶然机会触发犯罪 | 临时起意 |

## 五、凶手视角写作技巧

### 5.1 核心魅力

让读者进入"共犯"状态——知道真相却无法阻止。

### 5.2 关键要点

1. 让凶手有"合理"动机
2. 展现内心冲突
3. 避免过度美化
4. 展示心理变化过程

## 六、刑侦推理场景设计

```
1. 现场勘查 → 发现异常行为 → 推断心理状态
2. 受害者研究 → 分析选择模式 → 锁定嫌疑人特征
3. 行为模式比对 → 串并案件 → 预测下次作案
4. 侧写报告 → 缩小范围 → 引导侦查方向
```

---

## 七、核心公式

```
犯罪动机强度 = 需求迫切度 × 机会可获得性 × 道德约束力

罪犯画像准确度 = 行为证据 × 心理模式匹配 × 经验判断

角色可信度 = 动机合理 × 背景充分 × 行为一致
```
