# AI写小说：DeepSeek题材指令模板

通过结构化指令框架、多维度参数控制及跨题材适配策略，为创作者提供可复用的高效生成方案。

---

## 一、指令模板设计核心原则

在DeepSeek的小说生成场景中，指令模板需兼顾**创作自由度**与**输出可控性**。

### 分层指令架构

将创作任务拆解为四个层级：
1. 世界观设定
2. 人物设计
3. 情节推进
4. 细节描写

每个层级对应独立的指令模板。

**世界观指令模板示例**：
```
题材类型：东方玄幻
核心冲突：修仙门派与妖族的千年恩怨
地理特征：悬浮仙岛群+地脉灵泉网络
修炼体系：九重天阶（含隐藏第十重）
```

### 动态参数控制

引入可调节参数实现内容定制：

```
# 人物性格参数示例
character = {
    "主导特质": "外冷内热",
    "矛盾点": "守护宗门 vs 保护妖族恋人",
    "成长弧线": "从绝对正义到理解灰色地带"
}
```

### 多模态输出约束

结合文本描述与结构化数据：

```
# 战斗场景指令模板
视觉元素：紫电缠剑/青鳞显化
听觉元素：龙吟般的剑鸣
触觉反馈：剑柄传来的灼热感
时间维度：从日落到月升的三阶段变化
```

## 二、玄幻题材专用指令模板

玄幻小说创作需重点处理**修炼体系**、**法宝设定**和**势力格局**三大要素。

### 修炼体系生成指令

```
体系名称：太虚炼气诀
阶段划分：
1. 引气入体（特征：丹田气旋）
2. 凝脉成河（风险：经脉碎裂）
3. 筑基台成（标志：本命法宝孕育）
特殊机制：每突破一个大境界可解锁一种元素掌控
```

### 法宝设计指令

```
# 法宝属性生成器
artifact = {
    "名称": "九霄环佩琴",
    "品级": "地阶上品",
    "核心能力": "音波具现化",
    "限制条件": "需配合特定曲谱使用",
    "进化路径": "吸收天雷后可升级为雷音琴"
}
```

### 势力格局指令

```
正道联盟：
- 主导门派：太虚剑宗（剑修）
- 附属势力：炼丹师协会
- 隐藏矛盾：剑修与丹修的资源争夺

魔道联盟：
- 核心组织：血河宗（血修）
- 特殊能力：通过献祭增强修为
- 致命弱点：反噬风险随修为提升而加剧
```

## 三、言情题材专用指令模板

言情小说创作需强化**情感张力**、**人物互动**和**场景氛围**。

### 情感发展曲线指令

```
# 情感波动模型
emotion_curve = [
    {"阶段": "初遇", "强度": 0.3, "类型": "好奇"},
    {"阶段": "误会", "强度": 0.8, "类型": "痛苦"},
    {"阶段": "和解", "强度": 0.9, "类型": "释然"},
    {"阶段": "永恒", "强度": 1.0, "类型": "平静"}
]
```

### 亲密场景指令

```
场景类型：雨中告白
感官细节：
- 视觉：雨丝在灯光下呈银白色
- 触觉：湿透的衬衫贴合背部线条
- 对话设计：
  甲方："你明明知道..."
  乙方："知道什么？知道我会等你吗？"
隐喻系统：雨象征洗刷过去的阻碍
```

### 配角功能指令

```
# 配角角色卡
supporting_role = {
    "类型": "情敌",
    "威胁度": 0.6,
    "特殊技能": "制造巧合",
    "败露方式": "手机误发消息",
    "存在意义": "推动主角确认心意"
}
```

## 四、跨题材通用指令模板

### 情节转折生成器

```
转折类型：意外揭示
触发条件：主角发现家族秘籍中的隐藏批注
影响范围：
- 世界观：颠覆已知修炼体系
- 人物关系：导师的真实身份揭晓

```

### 对话系统指令

```

    "语境": "生死决战前夜",
    "潜台词": "我其实害怕失去你",
    "语言风格": "用玩笑掩盖真心",
    "非语言线索": "反复擦拭武器却始终不举"
```

```
情感升华：玉佩是主角母亲遗物
```

## 五、指令优化实践技巧

### 渐进式细化

先输入基础框架：
```
写一个现代都市异能故事
```

再逐步补充：
```
加入时间回溯能力
设定能力使用代价为记忆碎片丢失
```

### 反例验证法

通过故意设置矛盾指令测试系统：
```
让主角同时拥有最强攻击和最强防御
```

观察输出合理性并调整参数。

### 多版本对比

对同一情节使用不同指令模板生成，选择最优版本：

| 版本 | 方式 |
|------|------|
| A | 直接描述战斗结果 |
| B | 通过环境变化暗示胜负 |
| C | 用人物心理活动反映局势 |

### 结构化反馈

| 评估维度 | 评分标准 | 实际表现 |
|----------|----------|----------|
| 逻辑性 | 0-5分 | 4.2 |
| 创新性 | 0-5分 | 3.8 |
