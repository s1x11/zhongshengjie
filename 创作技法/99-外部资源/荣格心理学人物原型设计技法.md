```
┌─────────────────────────────────────┐
│          意识（自我Ego）             │ ← 人格最表层
├─────────────────────────────────────┤
```

## 二、四大核心原型

### 2.1 人格面具（Persona）

- **定义**：人格最外层，按他人期望行事的"假我"
- **功能**：适应社会，获得认可
- **危险**：过度认同面具导致失去真实自我
- **小说应用**：角色表面身份与内在真实的冲突

### 2.2 阴影（Shadow）

- **定义**：被压抑、拒绝的负面特质和欲望
- **特点**：既是破坏力源泉，也是创造力源泉
- **小说应用**：
  - 反派角色常是主角阴影的具象化
  - 角色突然失控行为（阴影浮现）
  - "光与影"主题的人物弧光

### 2.3 阿尼玛（Anima）与阿尼姆斯（Animus）

- **阿尼玛**：男性潜意识中的女性特质（情感、直觉）
- **阿尼姆斯**：女性潜意识中的男性特质（理性、逻辑）
- **小说应用**：
  - 男性角色遇见"梦中情人"（阿尼玛投射）
  - 女性角色遇见"灵魂导师"（阿尼姆斯投射）

### 2.4 自性（Self）

- **定义**：人格的中心和整体，包含意识与无意识
- **象征**：曼陀罗、圆形、神圣儿童、智者
- **小说应用**：角色的"天命召唤"时刻

## 三、十二原型系统

### 3.1 四大方向分类

```
┌─────────────────────────────────────────────────┐
│ 自我方向          │  秩序方向                     │
│ 英雄、魔术师、       │  看护者、创造者、              │
│ 亡命之徒           │  统治者                      │
├─────────────────────────────────────────────────┤
│ 社交方向          │  自由方向                     │
│ 情人、小丑、        │  天真者、圣人、               │
│ 普通人            │  探索者                      │
└─────────────────────────────────────────────────┘
```

### 3.2 小说创作七原型

| 原型 | 功能 | 代表角色 |
|------|------|----------|
| **英雄** | 故事主角，踏上旅程 | 哈利·波特、卢克·天行者 |
| **导师** | 提供智慧、训练、礼物 | 甘道夫、邓布利多 |
| **边界守卫** | 考验英雄，阻挡轻易进入 | 看门人、考官 |
| **信使** | 带来挑战信息，触发冒险 | 送信人 |
| **变形者** | 立场不明，制造悬疑 | 亦正亦邪的角色 |
| **阴影** | 反派，英雄的黑暗面对立面 | 伏地魔、达斯·维达 |
| **伙伴** | 陪伴英雄，提供帮助 | 罗恩与赫敏 |

## 四、象征符号系统

| 象征 | 含义 | 小说应用场景 |
|------|------|--------------|
| 曼陀罗 | 自性、完整、中心 | 角色顿悟时出现的圆形图案 |
| 水/海洋 | 无意识、情感、潜意识深度 | 跳入水中象征进入潜意识 |
| 蛇/龙 | 转化的力量、危险智慧 | 必须战胜的内心恶魔 |
| 阴影/黑暗 | 被压抑的内容、未知恐惧 | 地牢、洞穴、夜间森林 |
| 光明 | 意识觉醒、真理 | 山洞尽头的光、黎明时刻 |
| 阶梯/山峰 | 精神上升、成长历程 | 攀登高峰象征成长 |
| 镜像 | 自我认识、阴影投射 | 照镜子揭示真实自我 |

## 五、人物弧光设计

### 5.1 个体化进程三阶段

```
第一阶段：人格面具主导
├── 角色按社会期望行事
├── 压抑真实欲望和情感
└── 内心有空虚感、焦虑

↓ 触发事件 ↓

第二阶段：阴影浮现
├── 压抑的内容爆发
├── 角色失控、犯错误
└── 被迫面对黑暗面

↓ 整合工作 ↓

第三阶段：自性化
├── 接纳光明与黑暗
├── 整合阿尼玛/阿尼姆斯
└── "宁愿完整，不愿完美"
```

### 5.2 英雄之旅12阶段

```
1. 平凡世界 → 2. 冒险召唤 → 3. 拒绝召唤 → 4. 遇见导师 
→ 5. 跨越第一阈限 → 6. 考验/盟友/敌人 → 7. 深入洞穴 
→ 8. 严峻考验（死亡与重生）→ 9. 获得奖赏 
→ 10. 回归之路 → 11. 复活 → 12. 带着万灵丹回归
```

## 六、角色关系设计

```
角色关系动力学：

主角（Ego）────投射────→ 盟友（伙伴原型）
   │
   ├────投射────→ 导师（智慧老人=Self投射）
   │
   ├────投射────→ 爱人（阿尼玛/阿尼姆斯投射）
   │
   └────投射────→ 反派（阴影投射）
```

**高级技法**：
- 反派即主角阴影：反派拥有主角压抑的特质
- 爱人是灵魂镜像：爱人角色反映主角需要整合的内在异性特质
- 导师会死去：象征主角必须内化智慧

## 七、核心公式

```
角色完整度 = 阴影整合程度 × 阿尼玛/阿尼姆斯整合 × 人格面具灵活度

原型深度 = 核心特质 × 功能发挥 × 成长弧线

个体化进程 = 自我觉察 × 阴影面对 × 对立面整合
```

## 八、创作检查清单

```
□ 主角的阴影是什么？
□ 反派如何体现主角的阴影特质？
□ 主角需要整合哪些对立面？
□ 故事的象征系统是什么？
□ 角色关系是否反映原型投射？
□ 是否有完整的个体化弧线？
```
