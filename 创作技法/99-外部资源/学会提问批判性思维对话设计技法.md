# 学会提问批判性思维对话设计技法

> 来源：尼尔·布朗《学会提问》
> 整理时间：2026-03-31

---

## 一、理论来源

《学会提问》系统讲解批判性思维方法，核心命题：

> **批判性思维是提出正确问题的能力**

批判性思维框架可直接转化为角色对话设计、论辩场景设计、读者期待管理等创作技法。

---

## 二、两种思维方式

### 2.1 海绵式思维vs淘金式思维

| 类型 | 特点 | 小说应用 |
|------|------|----------|
| **海绵式思维** | 被动接收信息、强调知识获取 | 普通角色、被操控者 |
| **淘金式思维** | 主动提问、批判性评估 | 智者、侦探、主角成长目标 |

**小说转化**：
```
角色设计：
- 普通角色：海绵式思维，被动接受信息
- 成长型主角：从海绵式转向淘金式
- 智者型角色：始终保持淘金式思维

对话设计：
- 海绵式角色：少提问、多接受、易被骗
- 淘金式角色：多提问、质疑、验证

案例应用：
《三体》中的叶文洁vs罗辑：
叶文洁：海绵式接收人类文明负面信息→绝望
罗辑：淘金式思考黑暗森林法则→找到破局方法
```

### 2.2 弱势批判性思维vs强势批判性思维

| 类型 | 特点 | 小说应用 |
|------|------|----------|
| **弱势批判性思维** | 只批判他人、维护自己信念 | 偏执角色、固执角色 |
| **强势批判性思维** | 也批判自己、开放改变观点 | 真正的智者、成长型主角 |

**小说转化**：
```
角色成长路径：
第一阶段：海绵式思维（被动接受）
第二阶段：弱势批判性思维（批判他人）
第三阶段：强势批判性思维（自我批判）

角色对照：
- 弱势批判：反派、固执的配角
- 强势批判：智者、成长后的主角

案例应用：
《琅琊榜》梅长苏：始终保持强势批判性思维，但也在反思中成长
```

## 三、论题-论据-结论三角结构

### 3.1 核心结构

```
论题（讨论的问题）
    ↓
结论（作者的主张）
    ↑
论据（支持结论的证据）
```

**小说转化**：
```
对话场景设计：
- 论题：确定讨论的问题
- 结论：角色明确表达观点
- 论据：角色给出支撑证据

论辩结构：
角色A：论题→结论→论据
角色B：质疑论据/指出假设缺陷
角色A：补充论据/修正结论
```

### 3.2 寻找论题与结论

**识别方法**：

| 方法 | 说明 | 小说应用 |
|------|------|----------|
| **找论题指示词** | "是否"、"应不应该"、"……的问题" | 明确话题 |
| **找结论指示词** | "因此"、"所以"、"结论是"、"表明" | 明确观点 |
| **看开头结尾** | 结论常在开头或结尾 | 论辩对话结构 |

### 3.3 理由与证据类型

| 证据类型 | 特点 | 小说应用 |
|----------|------|----------|
| **直觉** | 个人感受 | 主观对话、情绪表达 |
| **个人经验** | 亲身经历 | 角色回忆、见证 |
| **典型案例** | 具体案例 | 举例论证 |
| **权威意见** | 专家观点 | 引用经典、师父教诲 |
| **研究结果** | 科学数据 | 数据论证 |
| **类比** | 相似性比较 | 类比论证 |

## 四、11个关键问题

### 4.1 关键问题清单

| 序号 | 问题 | 目的 | 小说应用 |
|------|------|------|----------|
| 01 | 论题和结论是什么？ | 明确讨论焦点 | 确定对话主题 |
| 02 | 理由是什么？ | 找到支撑证据 | 论辩场景设计 |
| 03 | 哪些词语意思不明确？ | 识别歧义 | 概念争论场景 |
| 04 | 什么是价值观假设和描述性假设？ | 发现隐含前提 | 揭示角色深层信念 |
| 05 | 推理过程中有无谬误？ | 检查逻辑错误 | 论辩取胜策略 |
| 06 | 这些证据的可信度有多大？ | 评估证据质量 | 质疑证据场景 |
| 07 | 有没有替代原因？ | 检查因果推理 | 推理情节设计 |
| 08 | 数据是否具有欺骗性？ | 检查数据有效性 | 数据欺骗情节 |
| 09 | 有什么重要信息被省略了？ | 检查信息完整性 | 信息隐瞒设计 |
| 10 | 能得出哪些合理的结论？ | 评估结论合理性 | 多种可能设计 |
| 11 | 这个结论与什么价值观冲突？ | 检查价值观立场 | 价值观冲突场景 |

### 4.2 深度解析关键问题

#### 问题03：哪些词语意思不明确？

**小说转化**：
```
概念争论场景：
- 关键词的歧义设计
- 角色对同一概念的不同理解
- 通过概念澄清揭示真相

案例应用：
《三体》：什么是"文明"？人类与三体的不同理解
《权力的游戏》：什么是"正义"？不同角色的价值观冲突
```

#### 问题04：什么是价值观假设？

**定义**：作者认为理所当然的价值观立场，未明说但影响推理。

**小说转化**：
```
角色深层动机设计：
- 表面论据 vs 隐含价值观
- 价值观冲突 = 角色冲突

价值观假设类型：
| 类型 | 定义 | 小说应用 |
|------|------|----------|
| **价值冲突** | 不同价值观的对立 | 角色价值观冲突 |
| **价值偏好** | 认为某种价值更重要 | 角色选择依据 |

案例应用：
《哈利波特》：斯内普的价值观假设——爱比恨更重要（直到结局才揭示）
```

#### 问题05：推理过程中有无谬误？

**常见谬误清单**：

| 谬误类型 | 定义 | 小说应用 |
|----------|------|----------|
| **人身攻击** | 攻击人而非论点 | 争吵场景 |
| **滑坡谬误** | 夸大连锁后果 | 恐吓、威胁 |
| **寻求完美方案** | 因方案不完美而否定 | 反驳场景 |
| **偷换概念** | 改变词义 | 诡辩话术 |
| **诉诸公众** | 多数人认为就对 | 从众压力 |
| **诉诸权威** | 权威说的就对 | 盲目服从 |
| **诉诸情感** | 用情感代替论证 | 情感操控 |
| **稻草人谬误** | 攻击歪曲的论点 | 恶意曲解 |
| **虚假两难** | 只给两个极端选择 | 逼迫选择 |
| **乱扣帽子** | 用名称代替解释 | 标签化 |

#### 问题07：有没有替代原因？

**小说转化**：
```
推理情节设计：
- 表面原因 vs 真实原因
- 单一原因 vs 多重原因
- 相关性 vs 因果性

替代原因类型：
| 类型 | 说明 | 小说应用 |
|------|------|----------|
| **多因一果** | 多个原因导致同一结果 | 复杂真相 |
| **共同原因** | C同时导致A和B | 揭示深层原因 |
| **因果倒置** | B导致A而非A导致B | 反转设计 |

案例应用：
《嫌疑人X的献身》：表面原因 vs 真实原因
```

#### 问题09：有什么重要信息被省略了？

**小说转化**：
```
信息隐瞒设计：
- 隐瞒关键证据
- 选择性披露信息
- 信息不对称制造悬念

信息省略类型：
| 类型 | 小说应用 |
|------|----------|
| **故意隐瞒** | 反派/智者的策略 |
| **无意遗漏** | 角色视角局限 |
| **选择性呈现** | 叙事操控 |

案例应用：
《罗生门》：同一事件的不同叙事，信息省略揭示人性
```

## 五、论辩场景设计框架

### 5.1 论辩结构设计

```
开场：
├── 明确论题
└── 提出立场

论证：
├── 理由1 + 证据
├── 理由2 + 证据
└── 理由3 + 证据

反驳预判：
├── 预测可能的反驳
└── 提前回应

收尾：
├── 重申立场
└── 呼吁行动
```

### 5.2 反驳策略设计

**基于11个关键问题的反驳**：

| 反驳策略 | 对应问题 | 应用方法 |
|----------|----------|----------|
| **质疑证据** | 问题06 | 指出证据不可靠 |
| **指出假设缺陷** | 问题04 | 揭示隐含假设问题 |
| **提出替代解释** | 问题07 | 提供不同因果链 |
| **指出信息遗漏** | 问题09 | 揭示被省略信息 |
| **指出推理谬误** | 问题05 | 揭示逻辑错误 |
| **质疑定义** | 问题03 | 指出关键词歧义 |

### 5.3 角色论辩风格

| 角色类型 | 论辩风格 | 特点 |
|----------|----------|------|
| **逻辑型** | 结构化论证 | 论据充分、逻辑严密 |
| **情感型** | 情感诉求 | 用故事、情感打动人 |
| **权威型** | 引用权威 | 引经据典、师出名门 |
| **诡辩型** | 制造谬误 | 故意混淆、转移话题 |
| **沉默型** | 用事实说话 | 行动胜于语言 |

## 六、对话设计技法

### 6.1 利用11个关键问题设计对话

**对话公式**：
```
角色A提出观点 → 角色B用关键问题质疑 → 角色A回应/修正

示例：
A："他是个好人，因为他经常捐款。"（论据）
B："捐款就是好人吗？他捐款的目的是什么？"（质疑定义+质疑假设）
A："他捐款是为了帮助别人。"（澄清）
B："你怎么知道他不是为了逃税或获取名声？"（质疑证据+替代动机）
```

### 6.2 价值观冲突对话设计

**设计公式**：
```
价值观A：认为X更重要
价值观B：认为Y更重要
冲突：X和Y不可兼得
对话：各自为自己的价值观辩护

案例应用：
《权力的游戏》：
小恶魔：智慧更重要
瑟曦：权力更重要
冲突：用权力还是智慧治理国家
```

### 6.3 批判性对话的戏剧效果

| 戏剧效果 | 设计方法 |
|----------|----------|
| **揭示真相** | 通过批判性提问逐步逼近真相 |
| **角色成长** | 学会用批判性思维打破偏见 |
| **情感冲击** | 质疑揭示角色深层动机 |
| **悬念制造** | 关键信息被省略，待后续揭示 |

## 七、实战应用清单

### 7.1 创作前：论辩规划

```
□ 确定论题：角色在争论什么？
□ 设计结论：各方观点是什么？
□ 设计论据：各方如何支撑观点？
□ 设计价值观假设：各方隐含什么价值观？
□ 规划反驳：角色如何反驳对方？
```

### 7.2 创作中：对话检查

```
□ 对话是否围绕明确论题？
□ 每个观点是否有论据支撑？
□ 是否有价值观冲突？
□ 是否有逻辑谬误（有意或无意）？
□ 是否有关键信息省略？
```

### 7.3 修订时：论辩审查

| 检查项 | 方法 |
|--------|------|
| 论题明确性 | 是否清楚角色在争论什么 |
| 论据充分性 | 每个结论是否有支撑 |
| 逻辑正确性 | 是否存在推理谬误 |
| 价值观合理性 | 角色价值观是否符合人设 |

## 八、核心公式总结

```
批判性思维质量 = 提问精准度 × 逻辑严密性 × 证据评估能力

论辩质量 = 论题明确度 × 论据充分度 × 推理正确度

海绵式→淘金式转化 = 被动接受 → 主动质疑 → 系统评估

弱势→强势批判性思维转化 = 批判他人 → 也批判自己

对话张力 = 价值观冲突强度 × 论据对抗强度 × 信息差
```
