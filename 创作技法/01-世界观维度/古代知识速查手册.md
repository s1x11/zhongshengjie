## 四、古代计时法

### 4.1 十二时辰制

| 时辰 | 别名 | 北京时间 |
|------|------|---------|
| **子时** | 夜半、子夜、中夜 | 23 时至 01 时 |
| **丑时** | 鸡鸣、荒鸡 | 01 时至 03 时 |
| **寅时** | 平旦、黎明、早晨、日旦 | 03 时至 05 时 |
| **卯时** | 日出、日始、破晓、旭日 | 05 时至 07 时 |
| **辰时** | 食时、早食 | 07 时至 09 时 |
| **巳时** | 隅中、日禺 | 09 时至 11 时 |
| **午时** | 日中、日正、中午 | 11 时至 13 时 |
| **未时** | 日昳、日跌、日央 | 13 时至 15 时 |
| **申时** | 哺时、日铺、夕食 | 15 时至 17 时 |
| **酉时** | 日入、日落、日沉、傍晚 | 17 时至 19 时 |
| **戌时** | 黄昏、日夕、日暮、日晚 | 19 时至 21 时 |
| **亥时** | 人定、定昏 | 21 时至 23 时 |

### 4.2 更点制度

**五更**：一夜分为五更，每更两个小时（120 分钟）

| 更次 | 时间 |
|------|------|
| **一更** | 19 时至 21 时 |
| **二更** | 21 时至 23 时 |
| **三更** | 23 时至次日凌晨 01 时 |
| **四更** | 01 时至 03 时 |
| **五更** | 03 时至 05 时 |

**每更五点**：每点 24 分钟

- **五更三点** = 早晨 5 时 72 分钟 = 6 时 12 分
- **三更四点** = 午夜 1 时 96 分钟 = 2 时 36 分
- **四更造饭，五更开船** = 后半夜 1 时至 3 时做饭，3 时至 5 时开船

### 4.3 百刻制度

**来源**：漏壶计时，可能起源于商代

**演变**：
- 汉代：120 刻
- 南朝梁：96 刻、108 刻
- 明末：提出 96 刻制改革
- 清初：定为正式制度

**换算**：一昼夜 100 刻 = 1440 分钟，每刻 14.4 分钟

**午时三刻** = 中午 1 时 43.2 分

### 4.4 日常时间称谓

| 称谓 | 时间 |
|------|------|
| **旦/早/朝/晨** | 日出时 |
| **夕/暮/晚** | 日入时 |
| **日中/正午/亭午** | 太阳正中时 |
| **隅中** | 将近日中时 |
| **昃/日昳** | 偏西时 |
| **黄昏** | 日入后 |
| **人定** | 黄昏后 |
| **夜半/夜分** | 人定后 |
| **鸡鸣** | 夜半后 |
| **昧旦/平明** | 鸡鸣后，天已亮 |

**用餐时间**：
- **食时/早食**：日出后隅中前（上餐）
- **晡时**：日昃后日入前（晚餐）
