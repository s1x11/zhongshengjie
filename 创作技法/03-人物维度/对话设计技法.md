## 一、对话的核心功能

### 1.1 对话的六大功能

| 功能 | 定义 | 占比建议 |
|------|------|----------|
| **推动情节** | 通过对话推进故事发展 | 30% |
| **塑造角色** | 通过对话展现角色性格 | 25% |
| **传递信息** | 通过对话传递必要信息 | 15% |
| **制造冲突** | 通过对话引发或升级冲突 | 15% |
| **营造氛围** | 通过对话建立场景氛围 | 10% |
| **揭示潜台词** | 通过对话暗示深层含义 | 5% |

### 1.2 对话质量公式

```
对话质量 = 推动情节力 × 角色个性化程度 × 信息传递效率 × 冲突张力 × 潜台词深度
```

## 二、潜台词设计

### 2.1 什么是潜台词

**定义**：角色所说的话与其真实意图之间的差距。

```
表面对话：角色说什么
潜台词：角色真正想表达什么

好的对话 = 表面对话 + 潜台词（两者不一致或有深意）
```

### 2.2 潜台词类型

| 类型 | 定义 | 示例 |
|------|------|------|
| **回避型** | 不直接回答问题 | "你还好吗？" "天快黑了。" |
| **暗示型** | 暗示真实意图 | "这里晚上不太安全。"（暗示离开） |
| **反讽型** | 说反话 | "你真是帮了大忙了。"（实际是添乱） |
| **转移型** | 转移话题 | "那件事...对了，你吃了吗？" |
| **沉默型** | 沉默本身就是潜台词 | 问："你爱他吗？" 答：（沉默） |

### 2.3 潜台词设计公式

```
潜台词深度 = |表面意思 - 真实意图| × 上下文丰富度

设计方法：
1. 确定角色真实意图
2. 设计表面对话
3. 在两者之间制造差距
4. 通过上下文暗示真实意图
```

**潜台词设计示例**：

```
场景：主角想借钱但不好意思开口

❌ 直白对话：
"能借我点钱吗？"
"可以，借多少？"

✅ 潜台词对话：
"最近手头有点紧..."
"又遇到什么麻烦了？"
"就是那个项目...需要一点启动资金。"
"你要多少？"
"也不多...够用就行。"
（潜台词：借钱的请求从未说出口，但双方都明白）
```

### 2.4 潜台词的层次

```
第一层：表面对话（字面意思）
第二层：情感潜台词（真实情感）
第三层：动机潜台词（深层动机）
第四层：主题潜台词（与主题的关联）

示例：
"你不必为我这么做。"（表面）
→ 我不希望你为我牺牲（情感）
→ 我害怕欠你的（动机）
→ 爱与自由的矛盾（主题）
```

## 三、角色对话风格设计

### 3.1 角色语言特征

**语言特征维度**：

| 维度 | 变化范围 | 示例 |
|------|----------|------|
| **句长** | 短句/长句 | 简洁vs啰嗦 |
| **用词** | 口语/书面 | "咋整"vs"如何处理" |
| **语气** | 肯定/犹豫 | 直接vs委婉 |
| **话题** | 自我/他人 | 以我为主vs关注他人 |
| **表达方式** | 直白/含蓄 | 有什么说什么vs绕弯子 |

### 3.2 角色对话模板

**角色对话特征表**：

| 角色类型 | 句长 | 用词 | 语气 | 话题 | 表达 |
|----------|------|------|------|------|------|
| **武将型** | 短 | 口语、粗犷 | 肯定 | 行动 | 直白 |
| **谋士型** | 长 | 书面、精准 | 谨慎 | 分析 | 含蓄 |
| **少年型** | 短 | 口语、新词 | 活跃 | 自我 | 直白 |
| **贵族型** | 长 | 书面、优雅 | 傲慢 | 身份 | 含蓄 |
| **平民型** | 短 | 口语、方言 | 随和 | 生活 | 直白 |

### 3.3 角色差异化设计

**对话差异化公式**：
```
角色辨识度 = 语言特征差异 × 思维方式差异 × 价值观差异

检查方法：
遮住名字，能否通过对话判断是谁在说话？
```

**差异化示例**：
```
同一句话的不同表达：

武将："干就完了！"
谋士："此事需从长计议。"
少年："冲啊！怕什么！"
贵族："本座自有安排。"
平民："咱们合计合计？"
```

## 四、多角色对话设计

### 4.1 多角色对话结构

**三人以上对话设计原则**：

```
原则1：明确对话主导者
├── 每段对话有明确的主导者
├── 其他角色配合或对抗
└── 避免七嘴八舌没有重点

原则2：轮流发言有逻辑
├── 按话题相关性分配发言
├── 不按座位顺序机械分配
└── 每个人发言有明确目的

原则3：对话节奏有变化
├── 快速交锋vs缓慢对话
├── 集中讨论vs分散插话
└── 避免单调重复
```

### 4.2 多角色对话模式

| 模式 | 特点 | 适用场景 |
|------|------|----------|
| **主导-配合式** | 一人主导，他人配合 | 会议、宣布 |
| **对抗-辩论式** | 双方对抗，他人旁观 | 争论、辩论 |
| **散点式** | 多人各自发言 | 聚会、闲聊 |
| **接力式** | 话题在多人间传递 | 讨论、协商 |

### 4.3 多角色对话设计模板

**五人会议场景模板**：

```
场景结构：

开场（主导者发言）：
"今天召集大家，是为了商量那件事。"

观点A（支持者）：
"我觉得应该..."

反对（反对者）：
"我不这么认为。"

补充（中立者）：
"我觉得两位说的都有道理..."

转折（关键人物）：
"等等，我有个问题。"

总结（主导者）：
"好，我们总结一下..."

设计要点：
├── 每人发言有明确目的
├── 发言顺序服务于情节
├── 有人主导，有人配合，有人反对
└── 不超过3轮循环
```

### 4.4 多角色对话节奏控制

```
节奏变化技巧：

快节奏：
├── 短句为主
├── 快速切换
├── 打断、插话
└── 紧迫感

慢节奏：
├── 长句为主
├── 沉思、停顿
├── 完整表达
└── 沉重感

节奏交替：
快→慢→快（紧张→思考→行动）
慢→快→慢（铺垫→高潮→收尾）
```

## 五、对话冲突设计

### 5.1 对话冲突类型

| 冲突类型 | 定义 | 表现方式 |
|----------|------|----------|
| **利益冲突** | 利益对立 | 争夺、谈判、威胁 |
| **价值观冲突** | 信念对立 | 争论、说服、否定 |
| **信息冲突** | 信息不对称 | 试探、隐瞒、欺骗 |
| **情感冲突** | 情感矛盾 | 指责、回避、爆发 |
| **权力冲突** | 地位对立 | 命令、抵抗、妥协 |

### 5.2 对话冲突升级公式

```
冲突升级阶梯：

Level 1：隐含分歧（表面和谐）
"你真的这么想？"

Level 2：直接表达分歧
"我不同意你的看法。"

Level 3：情绪升级
"你这是什么态度？"

Level 4：攻击性语言
"你就是个自私的人！"

Level 5：关系破裂
"我们没什么好说的了。"
```

### 5.3 对话张力公式

```
对话张力 = 角色目标冲突 × 信息差 × 时间压力 × 情感投入

增加张力的方法：
├── 双方目标不可调和
├── 关键信息被隐瞒
├── 时间紧迫，无法拖延
└── 情感深厚，无法冷静
```

## 六、对话信息传递

### 6.1 信息传递原则

```
原则1：自然嵌入
❌ "我父亲是武林盟主，母亲是魔教圣女。"
✅ "父亲总说，母亲当年的事，他不后悔。"

原则2：分散传递
├── 不要一次性倾倒大量信息
├── 在多个对话中分散信息
└── 让读者自己拼凑

原则3：冲突中传递
├── 在争论中暴露信息
├── 在质问中揭示真相
└── 信息服务于情节
```

### 6.2 信息传递技巧

| 技巧 | 方法 | 示例 |
|------|------|------|
| **问答式** | 一问一答传递信息 | "他去哪了？""不知道。" |
| **争论式** | 在争论中暴露信息 | "你竟然帮着外人说话！" |
| **回忆式** | 通过回忆传递背景 | "记得那年..." |
| **误会式** | 误会揭示真相 | "原来你一直都知道！" |

## 七、对话格式规范

### 7.1 对话格式类型

```
类型1：标准对话
他说："走吧。"

类型2：动作+对话
他放下茶杯："走吧。"

类型3：对话+动作
"走吧。"他站起身。

类型4：嵌入动作
"走吧，"他看了窗外一眼，"天快黑了。"

类型5：无标签对话
"走吧。"
"等等。"
"怎么了？"
```

### 7.2 对话标签使用原则

```
原则1：避免重复"说"
❌ 他说...他说...他说...
✅ 他说...他低声道...他反问...

原则2：动作优于标签
❌ 他生气地说："滚！"
✅ 他把杯子摔在地上："滚！"

原则3：必要时省略标签
├── 两人对话，每轮换行，可省略标签
├── 角色声音独特，可省略标签
└── 上下文清晰，可省略标签
```

## 八、对话场景设计

### 8.1 对话场景类型

| 场景类型 | 特点 | 设计要点 |
|----------|------|----------|
| **对抗型** | 双方对立 | 冲突明确、节奏紧凑 |
| **合作型** | 双方协作 | 目标一致、分工配合 |
| **试探型** | 信息不对称 | 言外之意、步步紧逼 |
| **情感型** | 情感交流 | 氛围营造、情感表达 |
| **仪式型** | 正式场合 | 格式规范、礼仪体现 |

### 8.2 对话场景结构

```
对话场景三段式：

开场（引入）：
├── 建立场景
├── 角色入场
└── 开启话题

中段（展开）：
├── 核心对话
├── 冲突发展
└── 信息传递

结尾（收束）：
├── 达成结果
├── 留下悬念
└── 过渡下一场景
```

## 九、实战应用清单

### 9.1 创作前：对话规划

```
□ 确定对话目的：推动情节/塑造角色/传递信息？
□ 设计对话双方：性格、目标、关系？
□ 是否需要潜台词？深度如何？
□ 是否存在冲突？如何升级？
□ 对话风格是否匹配角色？
```

### 9.2 创作中：对话检查

```
□ 对话是否推动情节？
□ 角色语言是否有辨识度？
□ 潜台词是否清晰又不过于隐晦？
□ 多角色对话是否有主导者？
□ 冲突是否有层次？
□ 信息传递是否自然？
```

### 9.3 修订时：对话审查

| 检查项 | 标准 |
|--------|------|
| 角色辨识度 | 遮住名字能否辨认是谁 |
| 潜台词深度 | 表面与深层是否形成张力 |
| 冲突强度 | 是否推动情节或揭示角色 |
| 信息自然度 | 是否避免生硬倾倒 |
| 节奏舒适度 | 是否有快慢变化 |

## 十、核心公式总结

```
对话质量 = 情节推动力 × 角色个性化 × 信息效率 × 冲突张力 × 潜台词深度

潜台词深度 = |表面意思 - 真实意图| × 上下文丰富度

角色辨识度 = 语言特征差异 × 思维方式差异 × 价值观差异

对话张力 = 目标冲突 × 信息差 × 时间压力 × 情感投入

冲突升级 = 隐含分歧 → 直接表达 → 情绪升级 → 攻击语言 → 关系破裂
```
