# 人物情感与性格描写技法

通过情感词汇、性格类型、心理状态的精准描写，让角色内心丰富立体。

---

## 一、情感描写技法

### 1.1 情感分类词库

| 情感类型 | 核心词汇 | 延伸词汇 |
|---------|---------|---------|
| **欣喜** | 快乐、振奋 | 大喜过望、欢天喜地、心花怒放、欣喜若狂、扬眉吐气、春风得意 |
| **愤怒** | 恼怒、激怒 | 怒火中烧、怒不可遏、怒发冲冠、七窍生烟、气冲牛斗、义愤填膺 |
| **悲痛** | 悲伤、哀愁 | 心如刀割、痛不欲生、肝肠寸断、泪如泉涌、人琴俱亡、椎心泣血 |
| **惊惧** | 恐惧、惊慌 | 心惊胆战、魂飞魄散、毛骨悚然、草木皆兵、风声鹤唳、心胆俱裂 |
| **爱恋** | 留恋、亲密 | 依依不舍、一见钟情、青梅竹马、如胶似漆、心心相印、一见如故 |
| **憎恨** | 愤恨、仇视 | 不共戴天、恨之入骨、深仇大恨、咬牙切齿、眼中钉、血海深仇 |

### 1.2 情感层次表

**浅层情感**（即时反应）：惊讶、喜悦、不满

**中层情感**（持续状态）：忧虑、焦虑、眷恋、厌恶

**深层情感**（核心驱动）：爱、恨、恐惧、希望、绝望

### 1.3 情感描写案例

**喜悦**：
- 心花怒放、满心欢喜、归心似箭、心旷神怡
- 她惊喜地回过头，脸上充满光辉，一双水汪汪的大眼睛发光地盯着他的脸

**悲痛**：
- 她的眼睛，那双怨时有雾、喜时有光、恨时有云、悲时没有泪的眼睛
- 倩儿转动一下眼珠，慢慢地从她的眼角迸出两滴泪珠

**愤怒**：
- 他的两眼黑得发亮，锋利的目光，仿佛要把什么刺穿似的
- 他目光如刀，散发出杀意

## 二、性格描写技法

### 2.1 性格类型分类

| 性格类型 | 核心特征 | 代表词汇 |
|---------|---------|---------|
| **活泼开朗** | 外向、乐观 | 俏皮、敏捷、幽默风趣、天真烂漫、心直口快 |
| **诚实真诚** | 真实、坦率 | 心口如一、襟怀坦白、真心诚意、开诚布公、一诺千金 |
| **虚伪刁滑** | 虚假、算计 | 表里不一、口是心非、老奸巨滑、笑里藏刀、虚情假义 |
| **谦虚热情** | 谦和、温暖 | 虚怀若谷、平易近人、古道热肠、和颜悦色、无微不至 |
| **坚强果断** | 刚毅、决断 | 坚不可摧、宁死不屈、当机立断、斩钉截铁、百折不挠 |
| **善良温柔** | 温和、仁慈 | 心慈手软、与人为善、温柔敦厚、菩萨心肠、一团和气 |
| **凶恶暴躁** | 凶狠、急躁 | 心狠手辣、穷凶极恶、暴跳如雷、狼心狗肺、丧心病狂 |
| **拘谨软弱** | 内向、怯懦 | 小心翼翼、唯唯诺诺、如履薄冰、优柔寡断、低声下气 |
| **安静悲观** | 沉静、消极 | 心灰意冷、万念俱灰、随遇而安、听天由命、萎靡不振 |

### 2.2 性格展现方式

**直接描写**（少用）：他是个诚实的人

**行为展现**（推荐）：
- 他答应的事，就算天塌下来也会做到
- 他说了谎，脸涨得通红，手不自觉地攥紧衣角

**对话展现**（推荐）：
- "这事我不占你便宜。"他把多找的零钱推回去
- "你说的，我信。"他点头，没再多问

## 三、心理描写技法

### 3.1 心理状态词库

| 心理状态 | 词汇 |
|---------|------|
| **紧张焦虑** | 七上八下、忐忑不安、心神不定、心急如焚、坐立不安 |
| **犹豫纠结** | 心事重重、心乱如麻、百感交集、悲喜交加 |
| **决心坚定** | 归心似箭、心悦诚服、迫不及待 |

### 3.2 心理描写层次

**表层心理**（想法）：他在想...

**中层心理**（感受）：他感到...

**深层心理**（本能）：他的身体...

### 3.3 心理描写案例

**紧张**：
- 他心里像揣了只兔子，七上八下
- 她的手心满是冷汗，一颗芳心不由自主地颤抖

**犹豫**：
- 他站在原地，去也不是，留也不是
- 她心里有两个声音在打架，一个说去，一个说别去

**决心**：
- 他暗暗下决心，在众人面前，要赢得干脆利落
- 她咬紧牙关，不让自己发出一点声音

## 四、动作描写技法

### 4.1 动作分类词库

| 动作类型 | 词汇 |
|---------|------|
| **看** | 望、视、瞧、瞅、瞟、眺、瞄、盯、瞥、窥、凝视、张望、遥望、怒视、注视 |
| **听** | 倾听、聆听、窃听、侧耳细听、听得入迷 |
| **想** | 猜想、推想、设想、回想、遐想、沉思、冥思苦想、绞尽脑汁 |
| **哭** | 啜泣、抽泣、呜咽、痛哭、潸然泪下、泪流满面、声泪俱下、嚎啕大哭 |
| **笑** | 微笑、大笑、憨笑、苦笑、冷笑、讥笑、笑逐颜开、眉飞色舞 |
| **手臂动作** | 捉、抓、擒、搂、抱、推、拉、打、捧、托、举、劈、拿 |
| **脚腿动作** | 行、走、奔、跑、跳、踢、蹲、站、跋涉、徘徊、蹒跚、踉跄 |

### 4.2 动作描写原则

**具体化**：不说"他很难过"，说"他蹲在墙角，把头埋进膝盖里"

**连贯性**：动作要有先后顺序，符合逻辑

**个性化**：不同性格的人动作不同（急性子动作快，慢性子动作慢）

### 4.3 动作描写案例

**看**：
- 他两眼下的黑圈不但颜色像烟熏出来的，并且线形也像缭绕弯曲、引人思绪的烟篆
- 她看起人来，两眼总是打闪似的快，眼光里充满了热情与聪敏

**哭**：
- 她的眼睫毛完全盖下来了，接着，那睫毛就被水雾所湿透；再接着，有两颗大大的泪珠，就从那密密的睫毛中滚落了下来
- 他倒在地上，脸如死灰，两唇惨白，一点血色也没有

**笑**：
- 她笑了，圆圆的脸庞就像一盆盛开的金葵花
- 他嘴角边绽出两个小酒窝，花一样美

## 五、声音描写技法

### 5.1 人物声音分类

| 声音类型 | 词汇 |
|---------|------|
| **悦耳** | 清脆、甜美、尖细、脆若银铃、出谷黄莺、莺声燕语、婉转动人 |
| **响亮** | 洪亮、磁性、雄浑、清朗、高昂、高亢、字正腔圆、铿锵有力、声如洪钟 |
| **负面** | 嘶哑、阴冷、呜咽、抽泣、哽咽、尖叫、怒吼 |

### 5.2 声音描写案例

- 她的声音低柔如清风
- 他的声音像从喉咙深处挤出来的，嘶哑而冰冷
- 她笑起来，声音像银铃一样清脆

## 六、快速调用指南

### 6.1 描写顺序

```
1. 情感触发（什么事）
2. 内心反应（怎么想）
3. 身体表现（什么动作/表情）
4. 后续行为（做什么）
```

### 6.2 性格展现技巧

| 技巧 | 说明 | 示例 |
|------|------|------|
| **对比法** | 不同性格对同一事的反应 | 急性子立刻冲出去，慢性子先想想 |
| **细节法** | 用小动作展现性格 | 说话时不自觉地摸鼻子（心虚） |
| **对话法** | 通过说话方式展现 | 话多的人外向，话少的人内向 |

### 6.3 注意事项

| ❌ 禁止 | ✅ 替代 |
|--------|--------|
| 直接说"他很伤心" | 描写具体表现（低头、沉默、眼眶红） |
| 性格标签化（好人/坏人） | 展现性格的复杂性和多面性 |
| 情感变化无过渡 | 设置情感触发点和转变过程 |
