# 史诗级创作学习路径规划

基于《史诗级著作核心成功要素分析框架》的学习笔记体系。

---

## 一、学习路径设计原则

| 原则 | 说明 |
|------|------|
| **先框架后细节** | 先掌握整体框架，再深入各维度技法 |
| **先核心后辅助** | 世界观→剧情→人物为核心，其他维度为辅助强化 |
| **先理论后实践** | 每维度先学理论，再做案例分析，最后练习创作 |
| **先对照后自主** | 先对照经典作品学习，后自主创作验证 |

---

## 二、维度学习优先级

### 第一阶段：核心维度（必修）

| 优先级 | 维度 | 学习理由 | 预计学时 |
|--------|------|----------|----------|
| P0 | **世界观** | 史诗级基础，决定作品格局和真实感 | 20h |
| P0 | **剧情** | 史诗级骨架，多线交织和历史回响 | 15h |
| P0 | **人物** | 史诗级血肉，群像塑造和命运线 | 15h |
| P1 | **氛围/意境** | 史诗级灵魂，厚重感和史诗感来源 | 10h |
| P1 | **叙事** | 史诗级技法，时间折叠和视角切换 | 10h |

**第一阶段目标**：掌握史诗级作品的"骨架"构建能力。

---

### 第二阶段：强化维度（选修）

| 优先级 | 维度 | 学习理由 | 预计学时 |
|--------|------|----------|----------|
| P2 | **战斗/冲突** | 史诗级张力，有代价的战斗和格局影响 | 8h |
| P2 | **主题** | 史诗级深度，永恒议题和多主题交织 | 8h |
| P2 | **情感** | 史诗级共鸣，集体情感和历史创伤 | 8h |

**第二阶段目标**：强化作品的"张力"和"深度"。

---

### 第三阶段：精修维度（进阶）

| 优先级 | 维度 | 学习理由 | 预计学时 |
|--------|------|----------|----------|
| P3 | **读者体验** | 史诗级沉浸，让读者"出不来"的秘密 | 5h |
| P3 | **元维度** | 史诗级信念，作者信念和世界观自洽 | 5h |

**第三阶段目标**：精修作品的"沉浸感"和"信念感"。

---

## 三、各维度学习路径

```
┌─────────────────────────────────────────────────────────────┐
│ 世界观学习路径                                                │

│ 第1步：理论学习                                               │
```

### 人物维度学习路径

```
┌─────────────────────────────────────────────────────────────┐
│ 人物学习路径                                                  │
├─────────────────────────────────────────────────────────────┤
│ 第1步：理论学习                                               │
│   - 群像塑造：多主角/多POV技法                                │
│   - 独立命运线：配角不是工具人                                 │
│   - 道德灰色地带：立场分明而非正邪分明                         │
│   - 选择有代价：不可逆后果                                    │
│   - 敌人有立场：反派逻辑而非邪恶设定                           │
├─────────────────────────────────────────────────────────────┤
│ 第2步：案例分析                                               │
│   - 《冰与火之歌》：POV群像、道德灰色、选择代价                 │
│   - 《三体》：敌人（三体文明）的生存逻辑                       │
│   - 《魔戒》：波洛米尔的道德灰色                               │
│   - 《时光之轮》：配角独立命运线                               │
├─────────────────────────────────────────────────────────────┤
│ 第3步：练习创作                                               │
│   - 练习1：设计5个有独立命运线的配角                          │
│   - 练习2：设计道德灰色角色（有立场有缺陷）                     │
│   - 练习3：设计有立场和逻辑的反派                              │
│   - 练习4：设计角色选择及其不可逆后果                          │
└─────────────────────────────────────────────────────────────┘
```
